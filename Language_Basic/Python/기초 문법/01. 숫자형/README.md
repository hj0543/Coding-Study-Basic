# 1. 숫자형 (Number)

### 1. 정수형 (Integer)
#### 정수(Integer)는 양의 정수, 음의 정수, 0을 다룹니다.
```python
a = 123
a = -178
a = 0
```
#### 정수의 진법표현
```python
print(0b1010)  # 10 (2진수)
print(0o12)    # 10 (8진수)
print(10)      # 10 (10진수)
print(0xA)     # 10 (16진수)
```
---
### 2. 실수형 (Floating-point)소수점이 포함된 숫자를 말합니다.
```Python
a = 1.2
a = -3.45
```
---
### 3. 연산자 (Operators)
#### 3-1. 사칙연산
```Python
a = 3
b = 4

a + b  # 7  (더하기)
a - b  # -1 (빼기)
a * b  # 12 (곱하기)
a / b  # 0.75 (나누기)
```
```Python
result = 0b1010 + 0o12 + 10 + 0xA
print(result)  # 40
# 2진수, 8진수, 10진수, 16진수 모두 정수 타입으로 취급
```
#### 3-2. 제곱과 나눗셈 특수 연산 (**, %, //)
```Python
a = 3
b = 4

# x의 y제곱 (**)
a ** b  # 3의 4승 -> 81

# 나눗셈 후 나머지 반환 (%)
7 % 3   # 1
3 % 7   # 3

# 나눗셈 후 몫 반환 (//)
7 // 4  # 1
```
#### 3-3. 우선순위
> 1) 괄호 ()
> 2) 지수 **
> 3) 곱셈 *, 나눗셈 /, 몫 //, 나머지 %
> 4) 덧셈 +, 뺄셈 -
```Python
-2 ** 4 = -16
-(2 ** 4) = -16
(-2) ** 4 = 16
```
---
### 4. 복합 대입 연산자
> 변수의 값을 변경한 후 다시 그 변수에 대입할 때 사용합니다.   

| 연산자 | 설명 | 동등한 표현 (예시) |
|:---:|:---|:---|
| `+=` | 더하고 대입 | `a = a + 1` |
| `-=` | 빼고 대입 | `a = a - 1` |
| `*=` | 곱하고 대입 | `a = a * 1` |
| `/=` | 나누고 대입 | `a = a / 1` |
| `//=` | 몫을 대입 | `a = a // 1` |
| `%=` | 나머지를 대입 | `a = a % 1` |
```Python
a += 1  # a는 11이 됨
a -= 5  # a는 6이 됨 (11 - 5)
a *= 2  # a는 12가 됨 (6 * 2)
```
