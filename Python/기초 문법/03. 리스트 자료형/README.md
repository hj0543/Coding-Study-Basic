# 3. 리스트 자료형 (List)
여러 개의 값을 **순서대로 저장**하는, **변경 가능**한 시퀀스 자료형
> **기본 형태**
```python
odd = [1, 3, 5, 7, 9]
```
---
### 1. 리스트의 생김새
```python
a = []
b = [1, 2, 3]
c = ['Life', 'is', 'too', 'short']
d = [1, 2, 'Life', 'is']
e = [1, 2, ['Life', 'is']]
```
---
### 2. 인덱싱 (Indexing)
#### 0부터 시작하며, 원하는 값을 하나씩 꺼낼 때 사용합니다.

```Python
a = [1, 2, 3]
a[0]   # 1
a[0] + a[2] # 1 + 3 = 4
a[-1]  # 3 (맨 뒤의 값)

# 인덱싱으로 값 수정하기
my_list = [1, 2. 3. 4. 5]
my_list[1] = 'two'
print(my_list) # [1, 'two', 3, 4, 5] -> 정수를 문자열로 수정

# 중첩된 리스트 인덱싱 (Nested list)
b = [1, 2, 3, ['a', 'b', 'c']]
b[-1]     # ['a', 'b', 'c']
b[-1][0]  # 'a'

# 삼중 리스트 인덱싱
c = [1, 2, ['a', 'b', ['Life', 'is']]]
c[2][2][0] # 'Life'
```
---
### 3. 슬라이싱 (Slicing)
#### 나눌 범위를 지정하여 꺼냅니다. 
**[시작:끝:스텝] (끝 번호는 포함되지 않음)**

```Python

a = [1, 2, 3, 4, 5]
a[0:2] # [1, 2]

# 문자열 슬라이싱과 비교
s = "12345"
s[0:2] # "12"

# 슬라이싱으로 여러 값 한 번에 바꾸기
my_list = [1, 2. 3. 4. 5]
my_list[2:4] = ['three', 'four']
print(my_list) # [1, 2, 'three', 'four', 5] -> 정수를 문자열로 수정

# 끝까지, 혹은 처음부터
b = a[:2]  # [1, 2] (처음 ~ 2미만)
c = a[2:]  # [3, 4, 5] (2 ~ 끝까지)

# 중첩 리스트 슬라이싱
d = [1, 2, 3, ['a', 'b', 'c'], 4, 5]
d[3][:2] # ['a', 'b']

# 스텝 리스트 슬라이싱
a[::2] # [1, 3, 5]

# 음수 슬라이싱
a[::-1] # [5, 4, 3, 2, 1]

# 음수 슬라이싱은 독립시행
d[3][::1] = ['c', 'b', 'a']
```

### 4. 리스트 연산
```Python

a = [1, 2, 3]
b = [4, 5, 6]

# 더하기
print(a + b)  # [1, 2, 3, 4, 5, 6]

# 반복하기
print(a * 3)  # [1, 2, 3, 1, 2, 3, 1, 2, 3]

# 길이 구하기
len(a)  # 3

# 주의: 형(Type) 불일치 오류
# a[2] + "hi" -> 숫자 + 문자열이라 에러 발생
str(a[2]) + "hi" # '3hi' (형 변환 필요)
```
---
### 5. 리스트 수정과 삭제

```Python

# 수정
a = [1, 2, 3]
a[2] = 4
# 결과: [1, 2, 4]

# 삭제 (del 키워드)
a = [1, 2, 3]
del a[1]
# 결과: [1, 3]

# 범위 지정 삭제
b = [1, 2, 3, 4, 5]
del b[2:]
# 결과: [1, 2]
```

### 6. 리스트 관련 함수 (Methods)
```Python
# append (추가)
a = [1, 2, 3]
a.append(4)      # [1, 2, 3, 4]
a.append([5, 6]) # [1, 2, 3, 4, [5, 6]] 리스트 자체를 추가

# sort (정렬)
a = [1, 4, 3, 2]
a.sort() # [1, 2, 3, 4]
b = ['a', 'c', 'b']
b.sort() # ['a', 'b', 'c']

# reverse (뒤집기)
b.reverse() # ['c', 'b', 'a']

# index (위치 찾기)
a = [1, 2, 3]
a.index(3) # 2 (3은 2번 인덱스에 있음)
# a.index(0) -> 없으면 에러 발생

# insert (삽입)
a = [1, 2, 3]
a.insert(0, 4) # 0번 자리에 4를 넣어라 -> [4, 1, 2, 3]

# remove (제거)
a = [1, 2, 3, 1, 2, 3]
a.remove(3) # 첫 번째 '3'만 삭제 -> [1, 2, 1, 2, 3]

# pop (끄집어내기)
a = [1, 2, 3]
a.pop()  # 3 (맨 뒤의 값을 반환하고 삭제)
a.pop(1) # 2 (1번 인덱스의 값을 반환하고 삭제)

# count (개수)
a = [1, 2, 3, 1]
a.count(1) # 2 (1이 두 개 있음)

# extend (확장)
a = [1, 2, 3]
a.extend([4, 5]) # [1, 2, 3, 4, 5] (리스트 더하기와 동일 효과)
```
