# 8. 변수 (Variable)

### 1. 변수란?
> 파이썬에서 변수는 객체(Object)를 가리키는 **포인터(참조)**와 같습니다.  
> `a = [1, 2, 3]`이라고 할 때, `[1, 2, 3]`이라는 리스트 객체가 메모리에 생성되고, 변수 `a`는 그 메모리 주소를 가리킵니다.

```python
a = [1, 2, 3]

# 메모리 주소 확인 (id 함수)
id(a)  # 예: 4303029896 (실행할 때마다 다름)
```
---
### 2. 리스트 복사할 때 주의할 점 (중요 ⭐)  
> 리스트 a를 b에 대입하면(b = a), 값이 복사되는 것이 아니라 같은 메모리 주소를 공유하게 됩니다.  

```Python

a = [1, 2, 3]
b = a  # 주소값 복사 (얕은 복사 X, 참조 복사)

a[1] = 4
# a -> [1, 4, 3]
# b -> [1, 4, 3] (b도 같이 변함!)

# 주소 비교
a is b  # True (완전히 동일한 객체를 가리킴)
```
---
### 3. 리스트 제대로 복사하기 (값이 같고 주소는 다르게)  
> 원래 리스트에는 영향을 주지 않고 복사본을 만들려면 아래 방법들을 사용해야 합니다.  

#### 3-1. 슬라이싱 이용하기 [:]  
리스트 전체 범위([:])를 슬라이싱하면 새로운 객체가 만들어집니다.  

```Python

a = [1, 2, 3]
b = a[:]  # 값만 복사해서 새로운 리스트 생성

a[1] = 4
# a -> [1, 4, 3]
# b -> [1, 2, 3] (b는 영향받지 않음)

a is b  # False (서로 다른 객체)
```

#### 3-2. copy 모듈 이용하기  
copy 모듈의 copy() 함수를 사용합니다.  

```Python

from copy import copy
a = [1, 2, 3]
b = copy(a)

b is a  # False
```
---
### 4. 변수 만드는 여러 가지 방법  
```Python

# 튜플로 한 번에 대입
a, b = ('python', 'life')
(a, b) = 'python', 'life'

# 리스트로 대입
[a, b] = ['python', 'life']

# 여러 변수에 같은 값 대입
a = b = 'python'

# 두 변수의 값 바꾸기 (Swap)
a = 3
b = 5
a, b = b, a  # a=5, b=3이 됨
```
