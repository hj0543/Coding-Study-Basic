# 6. 집합 자료형 (Set)

### 1. 집합이란?
> 집합(Set)은 파이썬 2.3부터 지원된 자료형으로, 수학의 집합과 동일하게 처리됩니다. `set()` 키워드를 사용해 만듭니다.
#### set 표현
```python
my_set_1 = set()
my_set_2 = {1, 2, 3}
my_set_3 = {1, 1, 1}

print(my_set_1)  # set() # 빈 집합
print(my_set_2)  # {1, 2, 3} # 순서 없음
print(my_set_3)  # {1} # 중복된 값은 하나로 처리됨
```
```python
# 리스트로 만들기
s1 = set([1, 2, 3])
# 결과: {1, 2, 3}

# 문자열로 만들기
s2 = set("Hello")
# 결과: {'e', 'H', 'l', 'o'}
```
---
### 2. 집합의 특징 (중요 ⭐)
> 집합 자료형에는 크게 두 가지 큰 특징이 있습니다.
*  중복을 허용하지 않습니다. (Unique)
*  순서가 없습니다. (Unordered) -> 인덱싱(s1[0]) 불가능.  (만약 인덱싱을 하려면 list()나 tuple()로 변환해야 합니다.)  
```Python
s1 = set([1, 2, 3])
l1 = list(s1)  # 리스트로 변환
l1[0]          # 1

t1 = tuple(s1) # 튜플로 변환
t1[0]          # 1
```
---
### 3. 교집합, 합집합, 차집합 구하기
> 수학 시간에 배운 집합 연산을 쉽게 처리할 수 있습니다.  

```Python

s1 = set([1, 2, 3, 4, 5, 6])
s2 = set([4, 5, 6, 7, 8, 9])

# 1. 교집합 (Intersection) &
s1 & s2              # {4, 5, 6}
s1.intersection(s2)  # {4, 5, 6}

# 2. 합집합 (Union) |
s1 | s2              # {1, 2, 3, 4, 5, 6, 7, 8, 9}
s1.union(s2)         # {1, 2, 3, 4, 5, 6, 7, 8, 9}

# 3. 차집합 (Difference) -
s1 - s2              # {1, 2, 3}
s1.difference(s2)    # {1, 2, 3}
s2 - s1              # {8, 9, 7}
```
---
### 4. 집합 관련 함수 (Methods)
> 값을 추가하거나 제거할 때 사용합니다.  

#### 1) add: 값 1개 추가하기  

* s1.add(4)  

#### 2) update: 값 여러 개 추가하기  

* s1.update([4, 5, 6])  

#### 3) remove: 특정 값 제거하기  

* s1.remove(2)  

```Python

s1 = set([1, 2, 3])

# 값 1개 추가 (add)
s1.add(4)
# 결과: {1, 2, 3, 4}

# 값 여러 개 추가 (update)
s1.update([4, 5, 6])
# 결과: {1, 2, 3, 4, 5, 6}

# 특정 값 제거 (remove)
s1.remove(2)
# 결과: {1, 3, 4, 5, 6}
```
