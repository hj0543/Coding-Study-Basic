# 7. 불 자료형 (Boolean)

### 1. 불(Bool)이란?
> 참(True)과 거짓(False)을 나타내는 자료형입니다.  
> 파이썬에서는 **첫 글자를 반드시 대문자**로 작성해야 합니다.  

```python
a = True
b = False

type(a)  # <class 'bool'>
```
---
### 2. 조건문의 리턴값
> 비교 연산자나 논리 연산의 결과는 항상 불(Bool) 자료형으로 나옵니다.

```Python

1 == 1      # True
2 > 1       # True
2 < 1       # False
```
---
### 3. 자료형의 참과 거짓 (중요 ⭐)
> 파이썬에서는 문자열, 리스트, 튜플 등 비어있는 상태인지 아닌지에 따라 참/거짓이 나뉩니다.  

거짓(False)인 경우:  
* 문자열: "" (빈 문자열)  
* 리스트: [] (빈 리스트)  
* 튜플: () (빈 튜플)  
* 딕셔너리: {} (빈 딕셔너리)  
* 숫자: 0 (0이 아닌 숫자는 모두 True)  
* 기타: None  

#### 사용 예시 (while문):  

```Python

# a가 빈 리스트가 될 때까지 반복
a = [1, 2, 3, 4]
while a:
    print(a.pop())
```
---
### 4. 불 연산 (bool 함수)
> bool() 내장 함수를 사용하면 해당 자료형이 참인지 거짓인지 확인할 수 있습니다.  

```Python

# 문자열
bool('python')  # True (값이 있음)
bool('')        # False (비어 있음)

# 리스트
bool([1, 2, 3]) # True
bool([])        # False

# 숫자
bool(3)         # True
bool(0)         # False
```
