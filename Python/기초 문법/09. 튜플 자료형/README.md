# 4. 튜플 자료형 (Tuple)

### 1. 튜플이란?
> 여러 개의 값을 순서대로 저장하는 변경 불가능한 시퀀스 자료
> 리스트와 거의 비슷하지만, **값을 바꿀 수 없다(Immutable)**는 점이 가장 큰 차이입니다.
> 튜플의 불변 특성을 사용하여 내부 동작과 안전한 데이터 전달에 사용됨 (다중 할당, 값 교환, 함수 다중 반환 등)
> 리스트는 대괄호 `[]`를 쓰지만, 튜플은 소괄호 `()`를 사용합니다. -> 소괄호 생략가능

**기본 생김새:** `t1 = (1, 2, 3)`

```python
t1 = ()
t2 = (1,)           # ⚠️ 요소가 1개일 땐 반드시 후행쉼표(,)를 붙여야 함
t3 = (1, 2, 3)
t4 = 1, 2, 3        # 괄호 생략 가능
t5 = ('a', 'b', ('ab', 'cd'))
```
---
### 2. 튜플의 가장 큰 특징   
> 튜플의 요솟값은 삭제하거나 수정할 수 없습니다.
> 따라서 프로그램 실행 중에 값이 변하면 안 되는 데이터는 튜플을 사용해야 합니다.

```Python
t1 = (1, 2, 'a', 'b')

# 삭제 시도 -> 오류 발생
# del t1[0]  (TypeError)

# 수정 시도 -> 오류 발생
# t1[0] = 'c' (TypeError)
```
---
### 3. 튜플 다루기 (인덱싱, 슬라이싱)   
> 값을 바꿀 순 없지만, 값을 조회하는 인덱싱과 슬라이싱은 리스트와 똑같이 사용할 수 있습니다.

```Python

t1 = (1, 2, 'a', 'b')

# 1. 인덱싱 (Indexing)
t1[0]     # 1
t1[3]     # 'b'

# 2. 슬라이싱 (Slicing)
t1[1:]    # (2, 'a', 'b')
```
---
### 4. 튜플 연산하기   
> 튜플끼리 더하거나 반복하면 새로운 튜플이 만들어집니다. (원본이 바뀌는 게 아님)

```Python

t1 = (1, 2, 'a', 'b')
t2 = (3, 4)

# 1. 튜플 더하기
t1 + t2   # (1, 2, 'a', 'b', 3, 4)

# 2. 튜플 곱하기 (반복)
t2 * 3    # (3, 4, 3, 4, 3, 4)

# 3. 튜플 길이 구하기
len(t1)   # 4
```
