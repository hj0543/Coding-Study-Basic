# [Python] 함수 작성 규칙 및 Scope (global) 정리

### 1. 함수 이름 구성 요소 (Naming Convention)

파이썬에서 함수 이름은 가독성을 위해 **PEP 8** 스타일 가이드를 따르는 것이 권장됩니다.

* **소문자 사용:** 모든 문자는 소문자로 작성합니다.
* **스네이크 케이스 (snake_case):** 단어와 단어 사이는 언더바(`_`)로 연결합니다.
* **동사 + 명사 구조:** 함수는 '동작'을 수행하므로, `동사`로 시작하여 `목적어`를 포함하는 형태가 좋습니다.
* *Good:* `get_user_name()`, `calculate_total_price()`, `is_valid()`
* *Bad:* `user()`, `price()`, `func1()` (기능을 유추하기 어려움)



### 2. 함수의 기본 규칙

함수를 올바르게 실행하기 위해 반드시 지켜야 할 문법적 규칙입니다.

1. **선언 후 호출 (Define before Call)**
* 파이썬은 인터프리터 언어이므로, 코드가 위에서 아래로 실행됩니다. 따라서 함수를 호출하기 전에 반드시 함수가 정의되어 있어야 합니다.


2. **들여쓰기 (Indentation)**
* 함수에 포함되는 코드 블록은 반드시 들여쓰기(Tab 또는 Space 4칸)를 해야 합니다. 들여쓰기가 맞지 않으면 `IndentationError`가 발생합니다.


3. **반환값 (Return)**
* `return` 키워드를 만나면 함수는 즉시 종료되며 값을 반환합니다.
* `return`이 없거나 값 없이 `return`만 쓰면 `None`을 반환합니다.



### 3. 변수의 유효 범위 (Scope)와 `global` 키워드

변수가 어디서 생성되었느냐에 따라 접근할 수 있는 범위가 달라집니다.

#### 지역 변수 (Local Variable) vs 전역 변수 (Global Variable)

* **지역 변수:** 함수 **내부**에서 정의된 변수. 함수가 실행되는 동안에만 메모리에 존재하며, 함수 밖에서는 접근할 수 없습니다.
* **전역 변수:** 함수 **외부**에서 정의된 변수. 프로그램 전체에서 접근할 수 있습니다.

#### `global` 키워드 사용법

함수 내부에서 전역 변수의 값을 **조회(읽기)**하는 것은 가능하지만, 값을 **수정(쓰기)**하려면 `global` 키워드가 필요합니다.

```python
count = 0  # 전역 변수

def increase_count():
    # 함수 내부에서 전역 변수 'count'를 수정하겠다고 선언
    global count  
    count += 1
    print(f"함수 내부: {count}")

increase_count()
print(f"함수 외부: {count}")

# 출력 결과:
# 함수 내부: 1
# 함수 외부: 1

```

> **주의사항:** `global`을 남용하면 변수의 값이 어디서 변경되었는지 추적하기 어려워져 유지보수가 힘들어집니다. (스파게티 코드의 원인) 가능하면 **매개변수와 반환값(return)**을 사용하여 데이터를 주고받는 것이 좋습니다.
