
# 📝 [TIL] CSS - Flexbox (플렉스박스) 완벽 정리

> **"수직/수평 정렬의 고통은 끝났다. 1차원 레이아웃의 끝판왕!"**

## 1. Flexbox란?

요소들을 행(Row)이나 열(Column) 형태로 배치하는 **1차원 레이아웃 방식**입니다.
과거에 `float`이나 `inline-block`으로 억지로 여백을 맞추던 시절을 끝내고, 요소의 크기와 순서를 아주 유연(Flexible)하게 배치할 수 있도록 만들어졌습니다.

* **선언 방법**: 정렬하고 싶은 요소들의 **부모 요소**에 `display: flex;`를 주면 마법이 시작됩니다.

---

## 2. Flexbox를 이해하는 4가지 핵심 개념 ★★★

Flexbox를 다루려면 아래 4가지 용어와 축(Axis)의 개념을 머릿속에 그려야 합니다.

1. **Flex Container (부모)**: `display: flex;`가 선언된 부모 요소. 자식들을 어떻게 배치할지 지휘하는 사령관 역할을 합니다.
2. **Flex Item (자식)**: 컨테이너의 1차 직계 자식 요소들. 사령관의 명령에 따라 유연하게 줄어들거나 늘어납니다.
3. **Main Axis (주축)**: 아이템들이 배치되는 **기본 방향**입니다. (기본값: 가로 방향, 왼쪽 $\rightarrow$ 오른쪽)
4. **Cross Axis (교차축)**: 주축에 **수직으로 교차**하는 방향입니다. (기본값: 세로 방향, 위 $\rightarrow$ 아래)

> 💡 **주의**: 방향 속성(`flex-direction`)을 세로(`column`)로 바꾸면, 주축이 세로가 되고 교차축이 가로로 바뀝니다!

---

## 3. Flex Container (부모)에게 주는 속성

전체적인 대형을 맞추고 정렬하는 속성들입니다.

### ① 배치 방향과 줄바꿈

* **`flex-direction`**: 주축(Main Axis)의 방향을 결정합니다.
* `row` (기본값): 가로 배치 (좌 $\rightarrow$ 우)
* `column`: 세로 배치 (위 $\rightarrow$ 아래)


* **`flex-wrap`**: 아이템이 컨테이너 크기를 넘어갈 때 **줄바꿈**을 할지 결정합니다.
* `nowrap` (기본값): 줄바꿈 안 함. 억지로 찌그러뜨려서 한 줄에 우겨넣습니다.
* `wrap`: 자리가 부족하면 아래로 자연스럽게 줄바꿈합니다.



### ② 정렬 (Alignment) ★ 가장 많이 씀

* **`justify-content`**: **주축(Main Axis)**을 기준으로 정렬합니다. (가로 배치일 땐 가로 정렬)
* `flex-start` (기본), `flex-end` (끝으로), `center` (가운데 정렬)
* `space-between`: 양끝을 벽에 붙이고 여백을 균등하게 분배합니다.
* `space-around`: 각 아이템의 양옆에 균등한 여백을 줍니다.


* **`align-items`**: **교차축(Cross Axis)**을 기준으로 정렬합니다. 보통 한 줄 내에서의 수직 정렬을 담당합니다.
* `stretch` (기본값: 교차축 크기만큼 쫙 늘림), `center` (수직 중앙 정렬)


* **`align-content`**: `flex-wrap: wrap;`으로 인해 **여러 줄(Multi-line)**이 생겼을 때, 그 줄들 사이의 간격을 교차축 기준으로 정렬합니다. (한 줄일 땐 작동하지 않음)

---

## 4. Flex Item (자식)에게 주는 속성

사령관(부모)의 명령을 따르되, **"나만 특별하게 행동할래!"** 할 때 개별 자식에게 주는 속성입니다.

* **`align-self`**: 부모가 지정한 `align-items`를 무시하고, **해당 아이템 하나만 독자적으로** 교차축 정렬을 바꿉니다. (예: 남들은 다 위에 붙었는데 나만 바닥에 붙고 싶을 때)
* **`flex-basis`**: 아이템의 **기본 뼈대 크기**를 설정합니다. (`width`와 비슷하지만, 주축 방향에 따라 너비가 될 수도, 높이가 될 수도 있는 유연한 크기입니다.)
* **`flex-grow`**: 컨테이너에 남는 빈 공간이 있을 때, 이 아이템이 **얼마나 더 덩치를 키워서 공간을 차지할지 비율**을 정합니다. (기본값: 0, 안 늘어남)
* **`flex-shrink`**: 공간이 부족할 때 **얼마나 더 쪼그라들지 비율**을 정합니다. (기본값: 1, `0`을 주면 절대 찌그러지지 않고 원래 크기를 유지합니다.)
* **`order`**: HTML 코드를 수정하지 않고, **시각적으로 보여지는 순서**만 숫자로 바꿉니다. (기본값: 0, 숫자가 작을수록 앞에 배치됨)

## 5. 💡 실전 압축 꿀팁 (Shorthand)

실무에서는 코드를 줄이기 위해 **단축 속성**을 매우 자주 사용합니다.

### 1. `flex` (아이템 유연성 단축)

자식 요소에게 부여하며, `flex-grow`, `flex-shrink`, `flex-basis`를 한 번에 작성합니다.

```css
.item {
  /* grow(1) shrink(1) basis(200px) */
  flex: 1 1 200px; 
}

/* 자주 쓰는 꼼수: 남는 공간을 모두 내 것으로 꽉 채우기 */
.item-fill {
  flex: 1; /* flex: 1 1 0%; 와 동일하게 작동 */
}

```

### 2. `flex-flow` (방향/줄바꿈 단축)

부모 요소에게 부여하며, `flex-direction`과 `flex-wrap`을 한 번에 작성합니다.

```css
.container {
  /* 가로로 배치하고, 넘치면 줄바꿈해라 */
  flex-flow: row wrap; 
}

```

✅ **마무리 점검**
"가운데 정렬"을 완벽하게 하는 마법의 코드는 개발자 밈(Meme)으로도 유명합니다. 부모 요소에 아래 3줄만 적으면 자식 요소가 화면 한가운데에 완벽하게 오게 됩니다.

```css
.center-box {
  display: flex;
  justify-content: center; /* 수평(주축) 중앙 */
  align-items: center;     /* 수직(교차축) 중앙 */
}

```
