
# 📝 [TIL] 알고리즘 - 힙(Heap)과 우선순위 큐 완벽 정리

> **"수십만 개의 데이터 중 가장 큰 값(또는 가장 작은 값)을 가장 빠르게 뽑아내는 마법의 트리"**

## 1. 힙(Heap)이란?

완전 이진 트리(Complete Binary Tree)에 있는 노드 중에서 **키 값이 가장 크거나 가장 작은 노드를 $O(1)$ 만에 찾기 위해** 만든 특수한 자료구조다.

### 🌟 힙의 2가지 종류

1. **최대 힙 (Max Heap)**
* **조건**: `부모 노드의 키 값 >= 자식 노드의 키 값`
* **루트 노드**: 트리에 있는 모든 노드 중 **가장 큰 값**.


2. **최소 힙 (Min Heap)**
* **조건**: `부모 노드의 키 값 <= 자식 노드의 키 값`
* **루트 노드**: 트리에 있는 모든 노드 중 **가장 작은 값**.



### 🚨 주의: 이진 탐색 트리(BST)와의 결정적 차이점

* BST는 왼쪽 자식과 오른쪽 자식 간의 크기 관계(왼쪽 < 부모 < 오른쪽)가 엄격하다.
* 반면, **힙은 형제 노드(왼쪽 자식과 오른쪽 자식) 사이에는 대소 관계가 전혀 없다.** 오직 부모와 자식 간의 위아래 관계만 성립하면 된다.

---

## 2. 힙(Heap)이 되기 위한 필수 조건

트리를 보고 "이것은 힙이다/아니다"를 판별할 줄 알아야 한다. 다음 두 가지 조건을 모두 만족해야만 힙이라고 부를 수 있다.

1. **모양 조건**: 반드시 **완전 이진 트리(Complete Binary Tree)**여야 한다. (중간에 이빨이 빠진 노드가 있으면 안 됨)
2. **값 조건**: 부모가 자식보다 항상 크거나(최대 힙), 항상 작아야 한다(최소 힙).

---

## 3. 힙의 삽입 연산 (Insertion)

새로운 데이터가 들어왔을 때, 힙의 규칙을 깨지 않고 자리를 찾아가는 과정이다. (시간 복잡도: $O(\log N)$)

1. **말단에 삽입**: 일단 완전 이진 트리의 형태를 유지하기 위해, 트리의 **가장 마지막 자리**에 새로운 노드를 삽입한다.
2. **부모와 비교 (Upheap)**: 새로 들어온 노드가 부모 노드보다 크다면(최대 힙 기준), 부모와 자리를 바꾼다. (Swap)
3. **반복**: 이 과정을 자신이 부모보다 작아지거나, 루트 노드에 도달할 때까지 계속 거슬러 올라가며 반복한다.

---

## 4. 힙의 삭제 연산 (Deletion) ★

힙에서는 **오직 "루트 노드"에 있는 원소만을 삭제하고 반환**할 수 있다. (중간에 있는 값을 임의로 뺄 수 없음)

1. **루트 노드 삭제**: 가장 큰 값(또는 가장 작은 값)인 루트 노드를 빼낸다.
2. **말단 노드를 루트로 이동**: 빈 루트 자리에, 트리의 **가장 마지막 노드**를 가져와 채워 넣는다. (완전 이진 트리 모양 유지)
3. **자식과 비교 (Downheap)**: 새로운 루트가 자식들보다 작다면(최대 힙 기준), 자식들과 자리를 바꾼다.
* *주의: 자식이 둘이라면, 둘 중 **더 큰 자식**과 자리를 바꿔야 힙의 구조가 깨지지 않는다.*


4. **반복**: 자신이 자식들보다 커지거나, 리프 노드에 도달할 때까지 계속 밑으로 내려간다.

---

## 5. 힙을 이용한 최강의 응용: 우선순위 큐 (Priority Queue)

일반적인 큐(Queue)는 먼저 들어온 데이터가 먼저 나가는 구조(FIFO)다.
하지만 **우선순위 큐**는 들어온 순서와 상관없이, **우선순위가 가장 높은(키 값이 가장 크거나 작은) 데이터가 먼저 나가는** 자료구조다.

* **왜 배열이나 리스트를 안 쓸까?**
* 배열로 우선순위 큐를 만들면, 데이터를 넣을 때마다 정렬해야 하거나 뺄 때마다 전체를 뒤져야 하므로 $O(N)$의 시간이 걸린다.


* **힙(Heap)을 쓴다면?**
* 최대 힙/최소 힙을 사용하면 삽입할 때도 $O(\log N)$, 삭제해서 최우선순위를 뽑아낼 때도 $O(\log N)$ 만에 끝난다!



### 💡 파이썬 실전 코딩 팁 (`heapq` 모듈)

코딩 테스트에서 힙을 1차원 배열로 직접 구현할 수도 있지만, 파이썬에서는 강력한 내장 라이브러리인 `heapq`를 사용한다. (기본적으로 **최소 힙**으로 동작한다.)

```python
import heapq

# 1. 빈 힙(우선순위 큐) 생성
pq = []

# 2. 삽입 (O(log N))
heapq.heappush(pq, 5)
heapq.heappush(pq, 1)
heapq.heappush(pq, 3)

# 현재 힙 상태: [1, 5, 3] (가장 작은 1이 루트에 위치)

# 3. 삭제 및 반환 (최솟값 뽑기, O(log N))
min_value = heapq.heappop(pq)
print(min_value) # 1 출력

# 💡 팁: '최대 힙'을 만들고 싶다면 값을 넣을 때 음수(-)를 붙여서 넣고, 뺄 때 다시 음수를 붙이면 된다!

```
