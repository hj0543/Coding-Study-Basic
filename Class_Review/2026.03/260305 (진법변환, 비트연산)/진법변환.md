## 1. 파이썬 내장 함수를 활용한 변환 (초간단!)

파이썬은 진법 변환을 위한 강력하고 직관적인 내장 함수를 제공한다. 실무나 코딩 테스트에서 빠르게 진법을 변환해야 할 때 주로 사용한다.

### 1. 10진수를 2진수, 16진수로 변환

* `bin(숫자)`: 10진수를 2진수 문자열로 변환 (앞에 `0b`가 붙음)
* `hex(숫자)`: 10진수를 16진수 문자열로 변환 (앞에 `0x`가 붙음)

```python
decimal_num = 149

print(bin(decimal_num))  # 출력: 0b10010101
print(hex(decimal_num))  # 출력: 0x95

```

### 2. 2진수, 16진수를 10진수로 변환

* `int('문자열', 진법)`: 문자열 형태의 숫자를 지정한 진법으로 읽어서 10진수 정수로 변환해 준다.

```python
bin_str = '10010101'
hex_str = '95'

print(int(bin_str, 2))   # 출력: 149
print(int(hex_str, 16))  # 출력: 149

```

## 2. 정석 알고리즘으로 직접 구현하기 (Under the hood)

내장 함수가 마법처럼 값을 바꿔주지만, 컴퓨터 내부에서 실제로 일어나는 연산 과정을 이해하기 위해 수학적 원리를 코드로 직접 구현해 보았다.

### 1. 10진수 ➡️ 2진수 변환 알고리즘 (나눗셈과 나머지)

10진수 숫자를 2로 더 이상 나눌 수 없을 때까지(몫이 0이 될 때까지) 계속 나누고, 그때마다 생기는 **나머지를 리스트에 저장한 뒤 마지막에 역순으로 뒤집는** 것이 핵심이다.

```python
tar = 149
result = []

# 1. 몫이 0이 될 때까지 계속 나누기
while tar != 0:
    result.append(tar % 2)  # 나머지를 리스트에 추가
    tar //= 2               # 몫을 업데이트

# 2. 나머지를 역순으로 뒤집기 (밑에서부터 읽어 올라가는 원리)
result.reverse()

print(result) 
# 출력: [1, 0, 0, 1, 0, 1, 0, 1]

```

올려주신 사진의 코드처럼 **리스트(List)**와 **수학 연산(몫, 나머지, 거듭제곱)**을 활용하는 것이 컴퓨터 과학에서 배우는 가장 '정석적인 알고리즘'입니다. 자료구조의 동작 원리를 이해하는 데 아주 좋죠.

내장 함수나 파이썬 특유의 문자열 연산을 배제하고, 사진에서 보여주신 방식(`append`, `reverse`, `while` 루프)을 확장하여 3가지 변환 과정을 다시 자세히 설명해 드릴게요.

### 2. 10진수 ➡️ 16진수 변환 알고리즘(나눗셈 알고리즘)
**2진수도 같이 가능**   

**알고리즘 순서:**

1. 바꿀 숫자(target)가 0이 될 때까지 밑수(2 또는 16)로 계속 나눕니다.
2. 매번 나온 **나머지**를 리스트에 `append`로 차곡차곡 담습니다.
3. 16진수일 경우, 나머지가 10 이상이면 딕셔너리를 참조해 A~F로 바꿔서 담습니다.
4. 나눗셈이 모두 끝나면 리스트를 `reverse()`로 뒤집어줍니다.

```python
def dec_to_base_algo(tar, base):
    if tar == 0:
        return ['0']
    
    result = []
    # 16진수를 위한 나머지 변환 사전
    hex_map = {10: 'A', 11: 'B', 12: 'C', 13: 'D', 14: 'E', 15: 'F'}
    
    while tar != 0:
        remainder = tar % base
        
        # 16진수이고 나머지가 10 이상일 때 문자로 변환
        if base == 16 and remainder >= 10:
            result.append(hex_map[remainder])
        else:
            result.append(str(remainder)) # 0~9는 문자열로 변환하여 리스트에 추가
            
        tar //= base # 몫을 업데이트
        
    result.reverse() # 마지막에 순서를 한 번에 뒤집기
    return result

print(dec_to_base_algo(149, 2))  # 출력: ['1', '0', '0', '1', '0', '1', '0', '1']
print(dec_to_base_algo(255, 16)) # 출력: ['F', 'F']

```

### 3. 2진수, 16진수 ➡️ 10진수 (자리수 가중치 알고리즘)

리스트에 담긴 2진수나 16진수 요소들을 차례대로 읽으면서, 해당 자리의 '가중치(밑수의 거듭제곱)'를 곱해 모두 더하는 방식입니다.

**알고리즘 순서:**

1. 1의 자리(오른쪽 끝)부터 계산하기 쉽도록 입력된 리스트를 먼저 한 번 뒤집습니다.
2. 인덱스(`i`)를 0부터 늘려가며 반복문을 돕니다. (이 `i`가 바로 지수가 됩니다)
3. 리스트의 값을 꺼내어 $값 \times Base^i$ 를 계산한 뒤 총합 변수에 더합니다.
4. 16진수 A~F 문자가 나오면 숫자로 바꿔서 계산합니다.

```python
def base_to_dec_algo(target_list, base):
    total_sum = 0
    # 계산의 편의를 위해 리스트 복사본을 만들어 뒤집음
    reversed_list = list(target_list)
    reversed_list.reverse()
    
    hex_map = {'A': 10, 'B': 11, 'C': 12, 'D': 13, 'E': 14, 'F': 15}
    
    # i는 0, 1, 2... 순서대로 증가하며 자리수를 나타냄
    for i in range(len(reversed_list)):
        char = reversed_list[i]
        
        # 16진수 문자열인 경우 숫자로 변환
        if base == 16 and char in hex_map:
            val = hex_map[char]
        else:
            val = int(char)
            
        # 가중치 계산: 값 * (밑수의 i제곱)
        total_sum += val * (base ** i)
        
    return total_sum

bin_list = ['1', '0', '0', '1', '0', '1', '0', '1']
hex_list = ['F', 'F']

print(base_to_dec_algo(bin_list, 2))  # 출력: 149
print(base_to_dec_algo(hex_list, 16)) # 출력: 255

```

### 4. 2진수 ↔️ 16진수 (4비트 그룹핑 알고리즘)

2진수 리스트를 4칸씩 잘라서 16진수 하나로 만들거나, 그 반대로 푸는 정석적인 과정입니다.

**알고리즘 순서 (2진수 ➡️ 16진수):**

1. 2진수 리스트의 전체 길이가 4의 배수가 되도록 리스트 맨 앞에 '0'을 `insert`로 채워 넣습니다.
2. 리스트를 4칸씩 건너뛰면서(슬라이싱) 4개의 숫자를 한 묶음(chunk)으로 가져옵니다.
3. 이 4개의 숫자를 각각 가중치( $2^3, 2^2, 2^1, 2^0$ )를 곱해 0~15 사이의 10진수로 만듭니다.
4. 계산된 값이 10~15라면 A~F로, 아니면 그대로 문자열로 바꾸어 결과 리스트에 `append` 합니다.

```python
def bin_to_hex_algo(bin_list):
    # 원본 보호를 위해 리스트 복사
    working_list = list(bin_list)
    
    # 1. 4의 배수가 되도록 리스트 맨 앞에 '0' 채우기
    while len(working_list) % 4 != 0:
        working_list.insert(0, '0')
        
    result = []
    hex_map = {10: 'A', 11: 'B', 12: 'C', 13: 'D', 14: 'E', 15: 'F'}
    
    # 2. 4칸씩 건너뛰며 반복
    for i in range(0, len(working_list), 4):
        chunk = working_list[i:i+4] # 4개짜리 작은 리스트 생성
        
        # 3. 4개의 2진수를 10진수 값으로 계산
        chunk_val = 0
        # 4자리 고정이므로 3, 2, 1, 0 제곱을 순서대로 곱함
        chunk_val += int(chunk[0]) * (2 ** 3)
        chunk_val += int(chunk[1]) * (2 ** 2)
        chunk_val += int(chunk[2]) * (2 ** 1)
        chunk_val += int(chunk[3]) * (2 ** 0)
        
        # 4. 10~15면 A~F로 변환하여 결과 리스트에 추가
        if chunk_val >= 10:
            result.append(hex_map[chunk_val])
        else:
            result.append(str(chunk_val))
            
    return result

sample_bin = ['1', '0', '1', '0', '1', '1', '1', '1']
print(bin_to_hex_algo(sample_bin)) # 출력: ['A', 'F']

```

### 💡 꿀팁: 리스트를 예쁜 문자열로 합치기 (`join`)

알고리즘을 통해 얻은 결과는 `[1, 0, 0, 1...]` 형태의 리스트다. 이를 내장 함수 `bin()`의 결과처럼 깔끔한 문자열 `10010101`로 만들고 싶다면, 파이썬의 문자열 메서드인 `join()`을 사용하면 된다.

*주의할 점: `join()`은 문자열만 합칠 수 있으므로, 리스트 안의 숫자를 문자로 먼저 바꿔주어야 한다.*

```python
# 리스트 안의 숫자(int)를 문자열(str)로 변환 (리스트 컴프리헨션 활용)
str_result = [str(x) for x in result]

# "".join()을 사용해 리스트의 요소들을 하나의 문자열로 이어 붙이기
final_string = "".join(str_result)

print(final_string)
# 출력: 10010101

```
