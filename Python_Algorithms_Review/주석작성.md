### 2. [구현 단계] '무엇(What)'보다 '왜(Why)'를 설명하기

* **매직 넘버 설명:** 왜 하필 `if x == 7:` 인지, 그 `7`이 의미하는 바를 적습니다.

### 🐍 실전 적용 예시: BFS 미로 찾기

```python
from collections import deque

'''
[문제 전략]
- 최단 거리 탐색이므로 BFS(너비 우선 탐색)를 선택
- 방문 배열을 따로 만들지 않고, graph 배열 자체에 거리를 누적하여 메모리 절약
- 시간 복잡도: O(N * M) - 격자의 모든 칸을 최대 한 번씩 방문
'''

def bfs(start_x, start_y, n, m, graph):
    """
    (start_x, start_y)에서 시작하여 (n-1, m-1)까지의 최단 거리를 계산
    """
    # 1. 큐 초기화 및 시작점 삽입
    queue = deque([(start_x, start_y)])
    
    # 이동 방향: 상, 하, 좌, 우
    dx = [-1, 1, 0, 0]
    dy = [0, 0, -1, 1]

    # 2. 큐가 빌 때까지 탐색 반복
    while queue:
        x, y = queue.popleft()

        # 현재 위치에서 4방향 확인
        for i in range(4):
            nx, ny = x + dx[i], y + dy[i]

            # 맵을 벗어나지 않고, 벽(0)이 아니며, 처음 방문(1)하는 칸인 경우
            if 0 <= nx < n and 0 <= ny < m and graph[nx][ny] == 1:
                # [의도] 현재 칸의 거리 = 이전 칸의 거리 + 1
                graph[nx][ny] = graph[x][y] + 1
                queue.append((nx, ny))
                
    # 도착 지점의 최단 거리 반환
    return graph[n-1][m-1]

# --- 메인 실행부 ---
# N, M 입력 및 그래프(미로) 정보 입력
N, M = map(int, input().split())
maze = [list(map(int, input().strip())) for _ in range(N)]

# (0, 0) 좌표에서 시작하여 결과 출력
result = bfs(0, 0, N, M, maze)
print(result)
```
