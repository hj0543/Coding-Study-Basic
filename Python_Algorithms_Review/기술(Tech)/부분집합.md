부분집합(Power Set)은 어떤 집합에 포함되는 원소들을 선택하여 만들 수 있는 모든 가능한 조합을 의미한다. 원소의 개수가 $n$개일 때, 각 원소는 '포함된다' 혹은 '포함되지 않는다'의 2가지 선택지를 가지므로 전체 부분집합의 개수는 **$2^n$개**가 된다.

부분집합은 **[알고리즘 - 완전 탐색/백트래킹]**의 범주에 속하며, 구현 방식에 따라 파이썬 기초나 비트 연산 스킬로도 분류할 수 있다.

---

### 💻 부분집합 구현 3가지 방법

#### 1. 재귀(DFS)를 이용한 방식 (가장 근본적인 알고리즘)

각 원소를 포함할지 말지 결정하며 깊게 파고드는 방식이다. 특정 조건에 맞지 않으면 탐색을 중단하는 '가지치기'를 적용하기에 가장 좋다.

```python
arr = [1, 2, 3]
n = len(arr)
visited = [False] * n

def get_subsets(depth):
    # 모든 원소의 포함 여부를 결정했다면 출력
    if depth == n:
        print([arr[i] for i in range(n) if visited[i]])
        return

    # 현재 원소를 포함하는 경우
    visited[depth] = True
    get_subsets(depth + 1)

    # 현재 원소를 포함하지 않는 경우
    visited[depth] = False
    get_subsets(depth + 1)

get_subsets(0)

```

#### 2. 비트마스킹을 이용한 방식 (파이썬 스킬 & 최적화)

이진수의 각 비트를 '원소의 포함 여부'로 매핑하는 천재적인 방법이다. 코드가 간결하고 실행 속도가 가장 빠르다.

```python
arr = [1, 2, 3]
n = len(arr)

# 1 << n은 2^n과 같음 (모든 경우의 수만큼 반복)
for i in range(1 << n):
    subset = []
    for j in range(n):
        # i의 j번째 비트가 1인지 확인
        if i & (1 << j):
            subset.append(arr[j])
    print(subset)

```

#### 3. itertools.combinations 활용 (실전 치트키)

0개부터 $n$개까지 뽑는 모든 조합을 합치면 부분집합이 된다. 단순히 모든 경우를 나열할 때 가장 편리하다.

```python
from itertools import combinations

arr = [1, 2, 3]
for r in range(len(arr) + 1):
    for comb in combinations(arr, r):
        print(list(comb))

```

### ⏱️ 주의 사항

부분집합의 개수는 $2^n$으로 지수적으로 증가한다. $n=20$일 때 약 100만 번의 연산이 필요하므로, 코딩 테스트에서 **$n$이 20을 넘어가는 경우에는 부분집합 완전 탐색을 사용하면 안 된다.**
