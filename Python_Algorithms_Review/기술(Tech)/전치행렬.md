**전치행렬(Transpose Matrix)**은 행렬의 행과 열을 서로 맞바꾸는 기법이다. 주대각선을 축으로 배열을 뒤집는다고 생각하면 이해가 빠르다. 90도 회전을 구현할 때나, 행 단위 연산을 열 단위로 처리해야 할 때 필수적으로 사용된다.

---

## 1. 전치의 원리 (Transpose Logic)

기본적으로 기존 배열의 $(i, j)$ 위치에 있던 원소는 전치 후 $(j, i)$ 위치로 이동한다.

* **기존:** `matrix[i][j]`
* **전치 후:** `transposed[j][i]`

---

## 2. 구현 방법 (3가지)

### ① 2중 for문을 이용한 정석적인 방법

새로운 $M \times N$ 크기의 배열을 만들고 값을 하나씩 옮겨 담는다. 인덱스 조작의 원리를 파악하기에 가장 좋다.

```python
def transpose_for(matrix):
    n = len(matrix)      # 기존 행 개수
    m = len(matrix[0])   # 기존 열 개수
    # m x n 크기의 빈 배열 생성
    result = [[0] * n for _ in range(m)]
    
    for i in range(n):
        for j in range(m):
            result[j][i] = matrix[i][j]
            
    return result

```

### ② zip과 언패킹(*)을 이용한 방법 (Pythonic)

파이썬 내장 함수인 `zip`을 활용하면 단 한 줄로 전치가 가능하다. 가장 널리 쓰이는 테크닉이다.

* `*matrix`: 각 행을 개별 인자로 푼다.
* `zip(...)`: 각 행의 같은 인덱스 요소들을 묶어 튜플로 만든다.

```python
def transpose_zip(matrix):
    # zip의 결과는 튜플이므로 리스트로 변환해준다
    return [list(row) for row in zip(*matrix)]

# 테스트
matrix = [[1, 2, 3], [4, 5, 6]]
# 결과: [[1, 4], [2, 5], [3, 6]]

```

### ③ NumPy를 이용한 방법

데이터 분석 환경이나 성능이 중요한 경우 NumPy의 `.T` 속성이나 `transpose()` 함수를 사용한다.

```python
import numpy as np
arr = np.array([[1, 2, 3], [4, 5, 6]])
transposed = arr.T  # 또는 arr.transpose()

```

---

## 3. 전치행렬의 실전 활용

1. **90도 회전 만들기:** * **시계 방향:** 전치(Transpose) 후 각 행을 뒤집기(Reverse) → `[row[::-1] for row in zip(*matrix)]`
* **반시계 방향:** 각 행을 뒤집기(Reverse) 후 전치(Transpose) → `list(zip(*[row[::-1] for row in matrix]))`


2. **열 단위 데이터 처리:** 파이썬 리스트는 행 단위 접근은 쉽지만 열 단위 접근은 어렵다. 이때 전치를 한 번 해주면 열 데이터가 행이 되어 접근이 매우 쉬워진다.
3. **대칭 행렬 확인:** `matrix == transpose(matrix)` 인지 확인하여 해당 행렬이 대칭(Symmetric)인지 판단할 수 있다.

---

**정리하자면**, 전치는 **"가로 데이터를 세로로 세우는 작업"**이다. `zip(*matrix)` 형태는 파이썬에서 매우 자주 쓰이는 패턴이므로 반드시 눈에 익혀두는 것이 좋다.
