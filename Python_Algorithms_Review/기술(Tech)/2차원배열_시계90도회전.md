2차원 배열을 시계 방향으로 90도 회전하는 두 가지 핵심 방법을 정리한다. 하나는 인덱스를 직접 계산하는 정석적인 방식이고, 다른 하나는 파이썬의 기능을 극대화한 방식이다.

---

## 1. 2중 for문을 이용한 정석적인 방법

이 방식은 새로운 배열을 미리 만들어두고, 기존 배열의 원소를 하나씩 적절한 위치에 꽂아 넣는 방식이다. 행과 열의 인덱스 변화 규칙을 이해하는 것이 핵심이다.

### 💡 회전 로직

* 기존 배열의 크기가 $N \times M$일 때, 회전 후 배열은 $M \times N$이 된다.
* 기존 배열의 좌표 $(i, j)$는 회전 후 $(j, N - 1 - i)$가 된다.
* **기존의 열($j$)**은 새로운 **행**이 된다.
* **기존의 행($i$)**은 역순으로 계산되어 새로운 **열**이 된다.



### 코드 구현

```python
def rotate_90_for(matrix):
    n = len(matrix)     # 행의 개수
    m = len(matrix[0])  # 열의 개수
    # 결과를 담을 M x N 크기의 빈 배열 생성
    result = [[0] * n for _ in range(m)]
    
    for i in range(n):
        for j in range(m):
            # 좌표 변환 공식 적용
            result[j][n - 1 - i] = matrix[i][j]
            
    return result

```

## 2. zip과 리스트 컴프리헨션을 이용한 방법

파이썬다운(Pythonic) 코드로, 단 한 줄로 해결 가능하다. `zip` 함수와 언패킹(`*`) 연산자의 특성을 활용한다.

### 💡 회전 로직

1. `matrix[::-1]`: 배열의 행 순서를 뒤집는다. (맨 아랫줄이 맨 위로 올라옴)
2. `*matrix[::-1]`: 뒤집힌 배열의 각 행을 인자로 낱낱이 푼다(Unpacking).
3. `zip(...)`: 인자로 들어온 행들의 같은 인덱스 요소끼리 묶는다.

### 코드 구현

```python
def rotate_90_zip(matrix):
    # 행을 뒤집고(reverse), 열끼리 묶어서(zip) 리스트로 만든다
    return [list(row) for row in zip(*matrix[::-1])]

```

---

## 🛠️ 두 방식의 한눈에 비교

| 구분 | 2중 for문 방식 | zip 리스트 컴프리헨션 |
| --- | --- | --- |
| **속도** | 평이함 | 파이썬 내장 함수라 상대적으로 빠름 |
| **가독성** | 인덱스 계산 로직이 보임 | 매우 간결하지만 원리 이해가 필요함 |
| **범용성** | C++, Java 등 타 언어에서도 동일 적용 | 파이썬 전용 |

---

**요약하자면**, 알고리즘 테스트에서 인덱스 조작 능력을 보여줘야 한다면 **for문**을, 실전에서 빠르게 기능을 구현해야 한다면 **zip** 방식을 사용하는 것이 유리하다.

혹시 90도 회전 외에 **반시계 방향 회전**이나 **180도 회전** 코드도 필요하다면 바로 말해주겠다. 도와줄까?
