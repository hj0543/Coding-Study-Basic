**`heapq`** 모듈은 파이썬에서 **우선순위 큐(Priority Queue)** 알고리즘을 구현할 때 사용하는 내장 라이브러리다.

일반적인 큐나 스택이 '들어온 순서'대로 데이터를 처리한다면, 우선순위 큐는 들어온 순서와 상관없이 **'우선순위가 가장 높은(파이썬에서는 가장 값이 작은) 데이터'**부터 먼저 꺼내주는 자료구조다. 코딩 테스트에서 최솟값이나 최댓값을 반복적으로 빠르게 찾아야 할 때, 그리고 최단 경로를 구하는 **다익스트라(Dijkstra)** 알고리즘을 구현할 때 절대 빠질 수 없는 필수 모듈이다.

### 💡 핵심 기능 및 동작 과정

파이썬의 `heapq`는 기본적으로 **최소 힙(Min-Heap)** 형태로 동작한다. 즉, 부모 노드가 자식 노드보다 항상 작거나 같은 완전 이진 트리 구조를 유지하며, **항상 0번째 인덱스(`heap[0]`)에 가장 작은 값이 위치**하게 된다.

1. **`heapq.heappush(heap, item)`:** 힙의 형태를 유지하면서 `item`을 추가한다.
2. **`heapq.heappop(heap)`:** 힙에서 가장 작은 원소(루트 노드)를 빼내어 반환한다. 빼낸 후에도 힙의 형태를 스스로 재정렬하여 유지한다.
3. **`heapq.heapify(list)`:** 이미 데이터가 들어있는 일반 리스트를 즉각적으로 힙 구조로 변환한다.

> **🔥 최대 힙(Max-Heap)을 만드는 꿀팁:**
> 파이썬은 최소 힙만 지원하므로, 최댓값을 먼저 뽑아내고 싶다면 데이터를 넣을 때 **부호를 반대로(`-`)** 바꿔서 넣으면 된다. 꺼낼 때 다시 부호를 원래대로 돌려주면 완벽하게 최대 힙으로 동작한다. 또는 `(-item, item)` 형태의 튜플로 넣으면 맨 앞의 값을 기준으로 정렬되는 튜플의 특성을 이용할 수 있다.

---

### 💻 heapq 기본 코드 (Python)

```python
import heapq

# 1. 최소 힙 사용하기
min_heap = []
heapq.heappush(min_heap, 5)
heapq.heappush(min_heap, 1)
heapq.heappush(min_heap, 3)

print(min_heap[0]) # 결과: 1 (가장 작은 값 확인, 꺼내지는 않음)
print(heapq.heappop(min_heap)) # 결과: 1 (가장 작은 값 꺼내기)
print(min_heap) # 결과: [3, 5]

# 2. 기존 리스트를 힙으로 변환하기
data = [4, 1, 7, 3, 8, 5]
heapq.heapify(data)
print(data) # 결과: [1, 3, 5, 4, 8, 7] (0번째 인덱스가 최솟값인 힙 구조로 변환됨)

# 3. 최대 힙(Max-Heap) 꼼수 사용하기
max_heap = []
items = [1, 3, 5, 7, 9]

# 값을 넣을 때 음수로 변환해서 push
for item in items:
    heapq.heappush(max_heap, -item)

# 꺼낼 때 다시 마이너스를 붙여서 원래 값으로 복구
print(-heapq.heappop(max_heap)) # 결과: 9 (가장 큰 값이 먼저 나옴)

```

### ⏱️ 시간 복잡도

* **삽입 (`heappush`):** **$O(\log N)$**
* **삭제 (`heappop`):** **$O(\log N)$**
* **변환 (`heapify`):** **$O(N)$**

리스트의 `min()`, `max()` 함수나 `sort()` 정렬을 사용하면 매번 $O(N)$이나 $O(N \log N)$의 시간이 걸리지만, 힙을 사용하면 매번 정렬할 필요 없이 단 **$O(\log N)$** 만에 최솟값/최댓값을 뽑아낼 수 있어 효율이 압도적이다.

### 📝 예시 문제: 최소 힙 (백준 1927번)

널리 잘 알려진 자료구조 중 최소 힙이 있다. 최소 힙을 이용하여 다음과 같은 연산을 지원하는 프로그램을 작성하시오.

1. 배열에 자연수 $x$를 넣는다.
2. 배열에서 가장 작은 값을 출력하고, 그 값을 배열에서 제거한다.
프로그램은 처음에 비어있는 배열에서 시작하게 된다.

* **입력:** 첫째 줄에 연산의 개수 $N$ ($1 \le N \le 100,000$)이 주어진다. 다음 $N$개의 줄에는 연산에 대한 정보를 나타내는 정수 $x$가 주어진다. 만약 $x$가 자연수라면 배열에 $x$라는 값을 넣는(추가하는) 연산이고, $x$가 0이라면 배열에서 가장 작은 값을 출력하고 그 값을 배열에서 제거하는 경우이다.
* **출력:** 입력에서 0이 주어진 횟수만큼 답을 출력한다. 만약 배열이 비어 있는 경우인데 가장 작은 값을 출력하라고 한 경우에는 0을 출력하면 된다.

---

### 💡 풀이 접근법

문제 이름 그대로 `heapq` 모듈을 가져다 쓰기만 하면 되는 가장 기초적인 문제다. 연산의 개수가 10만 개이므로 `sys.stdin.readline`을 사용해 입력 속도를 높이는 것이 필수다.

* 입력받은 수가 `0`이 아닐 때는 `heappush`로 힙에 넣는다.
* 입력받은 수가 `0`일 때는 힙이 비어있는지 확인한다.
* 비어있다면 `0`을 출력한다.
* 비어있지 않다면 `heappop`을 사용해 가장 작은 값을 꺼내어 출력한다.



---

### 💻 풀이 코드 (Python)

```python
import sys
import heapq
input = sys.stdin.readline

n = int(input())
heap = []

for _ in range(n):
    x = int(input())
    
    if x > 0:
        # 자연수면 힙에 추가
        heapq.heappush(heap, x)
    elif x == 0:
        # 0이 입력되었을 때
        if len(heap) == 0:
            # 힙이 비어있으면 0 출력
            print(0)
        else:
            # 힙이 비어있지 않으면 가장 작은 값 뽑아서 출력
            print(heapq.heappop(heap))

```
