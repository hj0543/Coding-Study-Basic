파이썬 3.6부터 도입된 **f-string (Formatted string literals)**은 파이썬 문자열 포맷팅의 **최종 진화 형태이자 절대 존엄**이다.

과거의 `%s`나 `.format()` 같은 번거롭고 길어지는 방식을 전부 관짝으로 보내버린, 코딩 테스트와 실무 모두에서 숨 쉬듯 써야 하는 압도적인 치트키다. 속도마저 기존 방식들보다 훨씬 빠르다.

### 💡 핵심 원리: 문자열 앞에 `f`만 붙이고, 변수는 `{}` 안에 넣는다!

그냥 문자열을 감싸는 따옴표 앞에 소문자 `f` (또는 대문자 `F`)를 적어주고, 문자열 중간에 변수나 값을 넣고 싶은 자리에 중괄호 `{}`를 열어서 때려 넣으면 끝이다.

---

### 💻 실전 활용 치트키 4가지 (Python)

코딩 테스트에서 무조건 마주치는 4가지 필수 포맷팅 패턴이다.

#### 1. 기본 변수 삽입

더 이상 문자열을 `+` 기호로 더하거나 콤마로 이어 붙일 필요가 없다.

```python
name = "다익스트라"
level = 99

# ❌ 과거의 끔찍한 방식
# print("내 이름은 " + name + ", 레벨은 " + str(level) + "이지.")

# ⭕ f-string의 우아한 방식
print(f"내 이름은 {name}, 레벨은 {level}이지.")
# 결과: 내 이름은 다익스트라, 레벨은 99이지.

```

#### 2. 중괄호 안에서 직접 연산 및 함수 호출하기

변수를 미리 가공할 필요 없이, `{}` 안에서 바로 수식 계산이나 함수 실행을 때려버릴 수 있다.

```python
a = 10
b = 3
arr = [1, 2, 3, 4, 5]

print(f"{a}와 {b}를 더하면 {a + b}입니다.")
# 결과: 10와 3를 더하면 13입니다.

print(f"이 배열의 길이는 {len(arr)}입니다.")
# 결과: 이 배열의 길이는 5입니다.

```

#### 3. 🔥 코테 단골: 숫자 포맷팅 (소수점, 자릿수, 콤마)

코딩 테스트에서 "소수점 넷째 자리에서 반올림하여 셋째 자리까지 출력하시오" 같은 악랄한 조건이 붙을 때 무조건 꺼내 써야 한다. 변수명 뒤에 `:`를 붙이고 형식을 지정한다.

```python
pi = 3.14159265
money = 10000000
num = 7

# 소수점 제한 (.nf 형식. 알아서 반올림까지 해준다!)
print(f"파이: {pi:.2f}") 
# 결과: 파이: 3.14

# 3자리마다 콤마(,) 찍기
print(f"내 전 재산: {money:,}원") 
# 결과: 내 전 재산: 10,000,000원

# 0으로 빈자리 채우기 (자릿수 맞추기)
print(f"요원 번호: {num:03d}") 
# 결과: 요원 번호: 007

```

#### 4. 디버깅의 신: `{변수명=}` (Python 3.8 이상 지원)

코딩 테스트 중에 변수에 값이 제대로 들어갔는지 빠르게 출력해 보고 싶을 때 쓰는 엄청난 꿀팁이다.

```python
target_x = 50
target_y = 100

print(f"{target_x=}, {target_y=}")
# 결과: target_x=50, target_y=100

```
