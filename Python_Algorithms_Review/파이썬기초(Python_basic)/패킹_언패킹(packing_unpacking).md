2차원 배열을 90도 회전시킬 때 썼던 `zip(*matrix)`에서 이 `*`(별표, Asterisk) 기호가 바로 언패킹의 핵심이다. 쉽게 비유하자면, 여러 개의 물건을 하나의 상자에 쓸어 담는 것이 **'패킹(포장)'**이고, 상자에 든 물건들을 와르르 쏟아서 각각의 자리에 예쁘게 배치하는 것이 **'언패킹(포장 풀기)'**이다.

---

### 💡 1. 기본 패킹과 언패킹

가장 기초적인 형태다. 파이썬은 쉼표(`,`)로 나열된 값들을 자동으로 튜플(Tuple)로 묶어주고(패킹), 변수의 개수만 맞으면 알아서 값을 쪼개서 넣어준다(언패킹).

```python
# 1. 패킹 (Packing)
# 괄호 없이 쉼표만 써도 파이썬이 알아서 하나의 튜플로 '포장'함
my_tuple = 1, 2, 3
print(my_tuple) # 결과: (1, 2, 3)

# 2. 언패킹 (Unpacking)
# 포장된 튜플을 개수에 맞춰 변수에 할당하면 알아서 '포장을 풀어줌'
a, b, c = my_tuple
print(a, b, c) # 결과: 1 2 3

# 🔥 코딩 테스트 치트키: 두 변수의 값 바꾸기 (Swap)
# 다른 언어처럼 임시 변수(temp)가 필요 없다.
a, b = 10, 20
a, b = b, a # b와 a를 패킹한 뒤, 다시 a와 b에 언패킹해서 넣음
print(a, b) # 결과: 20 10

```

### 💡 2. 별표(`*`)를 이용한 강력한 언패킹

코딩 테스트에서 데이터를 다룰 때 진가를 발휘하는 문법이다. 변수 앞에 `*`를 붙이면 **"남은 것들을 전부 리스트로 모아줘!"**라는 뜻이 된다.

```python
numbers = [1, 2, 3, 4, 5]

# 1. 첫 번째 값과 마지막 값만 빼고, 나머지는 묶어서 챙기고 싶을 때
first, *middle, last = numbers

print(first)  # 1
print(middle) # [2, 3, 4] (리스트로 패킹됨!)
print(last)   # 5

# 2. 리스트의 껍데기를 벗겨서 출력할 때 (for문 대체)
# 기존 방식: for n in numbers: print(n, end=' ')
print(*numbers) # 결과: 1 2 3 4 5 (대괄호 없이 알맹이만 출력됨)

```

### 💡 3. 함수의 `*args` 와 `**kwargs`

함수를 만들 때 인자(Argument)의 개수가 몇 개 들어올지 모른다면? 이때도 패킹이 쓰인다.

* **`*args` (Positional Arguments):** 들어오는 값들을 **튜플**로 묶어버린다 (별 1개).
* **`**kwargs` (Keyword Arguments):** `이름=값` 형태로 들어오는 값들을 **딕셔너리(해시)**로 묶어버린다 (별 2개).

```python
# 1. *args (여러 개의 숫자를 튜플로 패킹)
def add_all(*args):
    print("들어온 데이터:", args) # (1, 2, 3, 4) 튜플 형태
    return sum(args)

print(add_all(1, 2, 3, 4)) # 결과: 10


# 2. **kwargs (키워드 데이터를 딕셔너리로 패킹)
def print_info(**kwargs):
    print("들어온 데이터:", kwargs) # {'name': 'Gemini', 'age': 1} 딕셔너리 형태
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Gemini", age=1) 

```

---

패킹과 언패킹을 자유자재로 다루게 되면, 코드를 읽고 쓰는 속도가 차원이 다르게 빨라진다. 파이썬이 왜 코딩 테스트에서 가장 유리한 언어인지 보여주는 대표적인 기능이라 할 수 있다.

이제 이 탄탄한 기본기를 바탕으로, 유망하지 않은 길은 `continue`나 `return`으로 가차 없이 쳐내버리는 가지치기의 예술, **'백트래킹(Backtracking / N-Queen)'**으로 넘어갈까? 아니면 범위를 반씩 쪼개며 탐색하는 **'이진 탐색(Binary Search)'**의 템플릿부터 잡을까?
