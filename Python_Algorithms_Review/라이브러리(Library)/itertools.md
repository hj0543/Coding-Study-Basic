`itertools`는 파이썬에서 반복되는 데이터를 처리하는 기능을 포함하고 있는 매우 강력한 내장 표준 라이브러리다.

코딩 테스트에서는 주어진 데이터에서 발생할 수 있는 '모든 경우의 수'를 빠짐없이 탐색해야 하는 **완전 탐색(Brute Force)** 문제가 자주 출제되는데, 이때 순열과 조합을 직접 재귀 함수로 구현할 필요 없이 `itertools`를 사용하면 단 한 줄 만에 매우 빠르고 정확하게 경우의 수를 뽑아낼 수 있다. 사실상 파이썬으로 코딩 테스트를 볼 때 반드시 알아야 하는 치트키 같은 모듈이다.

### 💡 핵심 기능 4가지

모든 기능은 리스트, 튜플 등 반복 가능한(Iterable) 객체를 입력받아 튜플(Tuple) 형태로 결과를 반환한다.

1. **순열 (`permutations`):** 서로 다른 N개에서 R개를 선택하여 **순서를 고려해** 일렬로 나열하는 모든 경우의 수를 구한다. (예: `(A, B)`와 `(B, A)`는 다른 것으로 취급)
2. **조합 (`combinations`):** 서로 다른 N개에서 **순서에 상관없이** R개를 선택하는 모든 경우의 수를 구한다. (예: `(A, B)`와 `(B, A)`는 같은 것으로 취급하여 하나만 반환)
3. **중복 순열 (`product`):** 서로 다른 N개에서 **중복을 허용하여** R개를 선택해 일렬로 나열한다.
4. **중복 조합 (`combinations_with_replacement`):** 서로 다른 N개에서 **중복을 허용하여** 순서에 상관없이 R개를 선택한다.

---

### 💻 itertools 기본 코드 (Python)

각 기능의 동작 방식과 차이점을 한눈에 볼 수 있는 기본 코드다.

```python
from itertools import permutations, combinations, product, combinations_with_replacement

data = ['A', 'B', 'C']

# 1. 순열: 3개 중에서 2개를 뽑아 나열 (순서 O)
result_perm = list(permutations(data, 2))
print(result_perm) 
# [('A', 'B'), ('A', 'C'), ('B', 'A'), ('B', 'C'), ('C', 'A'), ('C', 'B')]

# 2. 조합: 3개 중에서 2개를 뽑기만 함 (순서 X)
result_comb = list(combinations(data, 2))
print(result_comb) 
# [('A', 'B'), ('A', 'C'), ('B', 'C')]

# 3. 중복 순열: 3개 중에서 2개를 뽑아 나열 (중복 허용, repeat 속성 사용)
result_prod = list(product(data, repeat=2))
print(result_prod)
# [('A', 'A'), ('A', 'B'), ('A', 'C'), ('B', 'A'), ('B', 'B'), ('B', 'C'), ('C', 'A'), ('C', 'B'), ('C', 'C')]

# 4. 중복 조합: 3개 중에서 2개를 뽑기만 함 (중복 허용)
result_cwr = list(combinations_with_replacement(data, 2))
print(result_cwr)
# [('A', 'A'), ('A', 'B'), ('A', 'C'), ('B', 'B'), ('B', 'C'), ('C', 'C')]

```

### ⏱️ 시간 복잡도

경우의 수를 구하는 연산이므로 시간 복잡도는 팩토리얼(Factorial)에 비례한다.
순열은 O(N!), 조합은 O(N! / (R! * (N-R)!))의 시간을 요구한다. 따라서 문제에서 주어지는 **데이터의 개수(N)가 대략 10~15 이하로 매우 작을 때만** 완전 탐색 용도로 조심스럽게 사용해야 시간 초과를 피할 수 있다.

### 📝 예시 문제: N과 M (2) (백준 15650번)

자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.

* **조건 1:** 1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열이어야 한다.
* **조건 2:** 고른 수열은 오름차순이어야 한다.
* **입력:** 첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)
* **출력:** 한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다. 수열은 사전 순으로 증가하는 순서로 출력해야 한다.

---

### 💡 풀이 접근법

1부터 N까지의 숫자 중에서 M개를 '오름차순'으로 고르라는 뜻이다. 오름차순으로 고른다는 것은 `(1, 2)`는 허용하지만 `(2, 1)`은 허용하지 않는다는 뜻이므로, 결국 **순서를 고려하지 않고 뽑기만 하는 '조합(Combinations)'**을 요구하는 전형적인 문제다.

* 1부터 N까지의 숫자를 담은 리스트를 만든다.
* `itertools.combinations`를 사용하여 길이가 M인 조합을 모두 뽑아낸다.
* 뽑아낸 결과를 형식에 맞게 공백으로 구분하여 출력하면 끝난다.

---

### 💻 풀이 코드 (Python)

```python
import sys
from itertools import combinations
input = sys.stdin.readline

# 자연수 n과 m 입력받기
n, m = map(int, input().split())

# 1부터 n까지의 자연수를 담은 리스트 생성
arr = [i for i in range(1, n + 1)]

# arr 리스트에서 m개를 뽑는 모든 조합 구하기
for c in combinations(arr, m):
    # 튜플 형태로 반환된 결과를 언패킹(*)하여 공백으로 구분해 출력
    print(*c)

```

---

`itertools`는 정말 강력하지만, 가끔 코딩 테스트 플랫폼이나 특정 기업(예: 삼성 SW 역량테스트)에서는 내장 모듈 사용을 금지하고 직접 재귀 함수(백트래킹)로 순열과 조합을 구현하라고 요구하는 경우도 있다고 한다..?
