트라이는 문자열을 저장하고 효율적으로 탐색하기 위한 트리(Tree) 형태의 자료구조다. 딕셔너리를 활용하면 특정 문자가 존재하는지 $O(1)$ 만에 확인할 수 있고, 단어의 끝을 표시하기 위해 별표(`*`) 같은 특수 문자를 키(Key)로 활용하여 구현을 매우 단순화할 수 있다.

### 💡 알고리즘 동작 과정

1. 빈 딕셔너리 `{}`를 생성하여 가장 바깥쪽의 루트 노드로 삼는다.
2. 문자열을 삽입할 때, 첫 번째 문자부터 순서대로 현재 딕셔너리의 키(Key)로 존재하는지 확인한다.
3. 만약 현재 문자가 키로 없다면, 해당 키의 값(Value)으로 **새로운 빈 딕셔너리를 생성하여 연결**한다.
4. 문자를 따라 딕셔너리 안쪽으로 계속 깊이 들어간다.
5. 문자열의 마지막 문자에 도달하면, 현재 위치한 딕셔너리에 **단어의 끝임을 알리는 특수 키(예: `'*'`)**를 삽입하여 플래그 역할을 하게 한다.
6. 탐색할 때도 마찬가지로 문자를 키로 삼아 딕셔너리를 파고들며, 중간에 키가 끊기거나 마지막 딕셔너리에 `'*'` 키가 없다면 찾는 단어가 트라이에 온전히 존재하지 않는 것이다.

---

### 💻 딕셔너리 활용 트라이 기본 코드 (Python)

클래스와 복잡한 변수 초기화 없이, 오직 딕셔너리와 함수만으로 동작하는 가벼운 코드다.

```python
# 빈 딕셔너리로 트라이 시작
trie = {}

# 1. 문자열 삽입 함수
def insert(trie, word):
    curr = trie
    for char in word:
        # 현재 문자가 딕셔너리에 없으면 새로운 빈 딕셔너리 생성
        if char not in curr:
            curr[char] = {}
        # 다음 딕셔너리(자식 노드)로 안으로 이동
        curr = curr[char]
    # 단어의 끝을 알리는 특수 키 '*' 삽입 (값은 True 등 아무거나 상관없음)
    curr['*'] = True

# 2. 접두사 탐색 함수 (해당 단어가 다른 단어의 접두사인지 확인)
def check_prefix(trie, word):
    curr = trie
    for char in word:
        # 단어가 채 끝나기도 전에 '*' 키가 있다면, 
        # 나보다 짧은 누군가의 단어가 이미 여기서 끝났다는 뜻 (접두사 발생)
        if '*' in curr:
            return False 
        curr = curr[char]
    return True # 무사히 끝까지 도달하면 접두사가 아님

```

### ⏱️ 시간 복잡도

트라이의 시간 복잡도는 탐색하려는 문자열의 길이를 $L$이라고 할 때, 삽입과 탐색 모두 **$O(L)$** 이다.
파이썬에서 딕셔너리의 키 탐색 시간 복잡도는 평균적으로 $O(1)$이므로, 단순히 문자열의 길이만큼만 반복문을 돌면 탐색이 끝난다. 전체 데이터의 개수와 무관하게 일정한 속도를 보장한다.

### 📝 예시 문제: 전화번호 목록 (백준 5052번)

전화번호 목록이 주어진다. 이때, 이 목록이 일관성이 있는지 없는지를 구하는 프로그램을 작성하시오. 전화번호 목록이 일관성을 유지하려면, 한 번호가 다른 번호의 접두어인 경우가 없어야 한다.
예를 들어, 구조대: 119, 지영석: 11 9552 4421 인 경우 구조대 번호가 지영석 번호의 접두어이므로 일관성이 없다.

* **입력:** 첫째 줄에 테스트 케이스의 개수 $t$ ($1 \le t \le 50$)가 주어진다. 각 테스트 케이스의 첫째 줄에는 전화번호의 수 $n$ ($1 \le n \le 10,000$)이 주어지고, 다음 $n$개의 줄에는 전화번호가 하나씩 주어진다. (전화번호의 길이는 최대 10자리)
* **출력:** 각 테스트 케이스에 대해서, 일관성 있는 목록인 경우에는 `YES`, 아닌 경우에는 `NO`를 출력한다.

---

### 💡 풀이 접근법

어떤 문자열이 다른 문자열의 **'접두사(Prefix)'**인지 확인하는 문제이므로, 방금 만든 딕셔너리 기반의 트라이를 활용하면 매우 쉽게 풀린다.

* 매 테스트 케이스마다 빈 딕셔너리 `trie = {}`를 만들고, 주어지는 모든 전화번호를 `insert` 함수를 통해 딕셔너리 안에 겹겹이 쌓아 넣는다.
* 다시 모든 전화번호를 순회하며 `check_prefix` 함수로 탐색을 진행한다.
* 탐색 도중 아직 내 번호가 끝나지도 않았는데 현재 딕셔너리 안에 `'*'` 키가 존재한다면, 일관성이 없는 것(`NO`)으로 판별하고 즉시 탐색을 종료하면 된다.

---

### 💻 풀이 코드 (Python)

```python
import sys
input = sys.stdin.readline

# 트라이에 문자열(전화번호)을 삽입하는 함수
def insert(trie, word):
    curr = trie
    for char in word:
        if char not in curr:
            curr[char] = {}
        curr = curr[char]
    curr['*'] = True

# 다른 번호의 접두사인지 확인하는 함수
def check_prefix(trie, word):
    curr = trie
    for char in word:
        # 내 번호를 마저 읽기도 전에 '*'이 있다면 누군가 여기서 번호가 끝났다는 뜻
        if '*' in curr:
            return False
        curr = curr[char]
    return True

t = int(input())
for _ in range(t):
    n = int(input())
    trie = {}      # 매 테스트 케이스마다 새로운 빈 딕셔너리로 초기화
    words = []
    
    # 1. 모든 전화번호를 입력받아 트라이에 삽입
    for _ in range(n):
        word = input().rstrip() # 줄바꿈 문자('\n') 제거
        words.append(word)
        insert(trie, word)
        
    is_consistent = True
    
    # 2. 삽입된 전화번호들을 다시 순회하며 일관성 확인
    for word in words:
        if not check_prefix(trie, word):
            is_consistent = False
            break # 하나라도 접두사가 겹치면 더 볼 필요 없이 탈출
            
    # 3. 결과 출력
    if is_consistent:
        print("YES")
    else:
        print("NO")

```
