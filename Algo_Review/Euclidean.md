유클리드 호제법(Euclidean Algorithm)은 두 자연수의 **최대공약수(GCD, Greatest Common Divisor)**를 아주 빠르고 우아하게 구해내는 기원전 300년경 고대 그리스의 수학 알고리즘이다.

단순히 1부터 차례대로 숫자를 나누어보며 최대공약수를 찾는 방식은 숫자가 커질수록 시간이 오래 걸리지만, 유클리드 호제법은 나눗셈의 **'나머지'**를 활용해 탐색 범위를 기하급수적으로 줄여버린다. 이 원리를 이용하면 두 수의 **최소공배수(LCM, Least Common Multiple)**도 공식 하나로 1초 만에 뚝딱 알아낼 수 있다.

### 💡 알고리즘 동작 과정

유클리드 호제법의 핵심 원리는 **"$A$를 $B$로 나눈 나머지를 $R$이라고 할 때, $A$와 $B$의 최대공약수는 $B$와 $R$의 최대공약수와 같다"**는 성질을 이용하는 것이다.

1. 두 자연수 `A`와 `B`가 주어졌을 때 (`A > B`라고 가정), `A`를 `B`로 나눈 나머지 `R`을 구한다. (`A % B = R`)
2. 만약 나머지 `R`이 0이라면, 나누어 떨어졌다는 뜻이므로 그 순간의 **`B`가 바로 두 수의 최대공약수**가 된다.
3. 만약 나머지 `R`이 0이 아니라면, `A` 자리에 `B`를 넣고, `B` 자리에 `R`을 넣은 뒤 다시 1번 과정을 반복한다.

> **참고:** 코드로 구현할 때는 처음에 `A`가 `B`보다 작아도 상관없다. 첫 번째 나눗셈(`A % B`)을 거치고 나면 두 수의 위치가 알아서 뒤바뀌기 때문이다.

### 💻 유클리드 호제법 기본 코드 (Python)

파이썬에서는 다중 할당(Multiple Assignment)을 지원하므로, 반복문(while)을 사용해 단 세 줄 만에 아주 깔끔하게 구현할 수 있다.

```python
# 최대공약수(GCD)를 구하는 함수
def gcd(a, b):
    # 나머지가 0이 될 때까지 반복
    while b != 0:
        # a 자리에 b를, b 자리에 a를 b로 나눈 나머지를 동시에 대입
        a, b = b, a % b
    
    # b가 0이 되어 반복문이 끝났을 때의 a가 최대공약수
    return a

# 최소공배수(LCM)를 구하는 함수
def lcm(a, b):
    # 두 수의 곱을 최대공약수로 나누면 최소공배수가 됨 (A * B / GCD)
    return (a * b) // gcd(a, b)

print(gcd(192, 162)) # 결과: 6

```

### ⏱️ 시간 복잡도

유클리드 호제법의 시간 복잡도는 **O(log N)** 이다. ($N$: 두 수 중 작은 수)
나머지를 구하는 과정을 한 번 거칠 때마다 숫자의 크기가 최소 절반 이하로 뚝뚝 떨어지기 때문에, 두 수가 수십억 단위로 아무리 커져도 연산 횟수는 수십 번을 넘지 않아 매우 압도적인 속도를 자랑한다.

### 📝 예시 문제: 최대공약수와 최소공배수 (백준 2609번)

두 개의 자연수를 입력받아 최대 공약수와 최소 공배수를 출력하는 프로그램을 작성하시오.

* **입력:** 첫째 줄에는 두 개의 자연수가 주어진다. 이 둘은 10,000 이하의 자연수이며 사이에 한 칸의 공백이 주어진다.
* **출력:** 첫째 줄에는 입력으로 주어진 두 수의 최대공약수를, 둘째 줄에는 최소공배수를 출력한다.

---

### 💡 풀이 접근법

문제 이름 그대로 유클리드 호제법을 사용해 최대공약수를 구하고, 이를 바탕으로 최소공배수를 계산하는 가장 기초적인 문제다.

* **최대공약수:** `while`문을 활용한 `gcd(a, b)` 함수로 구한다.
* **최소공배수:** 두 수 `A`와 `B`의 곱은 '최대공약수 × 최소공배수'와 같다는 수학적 성질(`A * B = GCD * LCM`)을 이용한다. 따라서 `(A * B) // GCD`를 계산하면 최소공배수가 나온다.

---

### 💻 풀이 코드 (Python)

```python
import sys
input = sys.stdin.readline

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

a, b = map(int, input().split())

# 1. 최대공약수 구하기
result_gcd = gcd(a, b)

# 2. 최소공배수 구하기 (두 수의 곱을 최대공약수로 나눈 몫)
result_lcm = (a * b) // result_gcd

# 결과 출력
print(result_gcd)
print(result_lcm)

```

> **꿀팁:** 사실 파이썬 코딩 테스트 실전에서는 직접 이 함수를 짤 필요 없이, 내장 모듈인 `math`를 불러와서 `math.gcd(a, b)`와 `math.lcm(a, b)`를 사용하면 단 한 줄로 끝낼 수 있다. 하지만 원리를 묻는 문제나 응용문제가 나올 수 있으므로 호제법의 원리는 반드시 알아두어야 한다.
