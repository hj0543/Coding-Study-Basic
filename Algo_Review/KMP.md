KMP(Knuth-Morris-Pratt) 알고리즘은 특정한 긴 텍스트(문자열) 안에서 짧은 패턴(문자열)이 등장하는지, 등장한다면 어느 위치에 있는지를 매우 빠르게 찾아내는 대표적인 **문자열 탐색 알고리즘**이다.

일반적으로 문자열을 탐색할 때 불일치가 발생하면 텍스트의 탐색 위치를 처음부터 다시 되돌려야 하지만(단순 비교의 경우), KMP 알고리즘은 **'지금까지 검사한 문자열의 접두사와 접미사 정보'**를 활용하여 텍스트의 탐색 위치를 뒤로 돌리지 않고 건너뛰며 탐색한다는 특징이 있다.

### 💡 알고리즘 동작 과정

1. 패턴 문자열을 분석하여 **LPS(Longest Prefix Suffix) 테이블**을 만든다. (이 테이블은 패턴의 각 인덱스까지의 부분 문자열에서, '접두사와 접미사가 일치하는 최대 길이'를 담고 있으며 **실패 함수**라고도 부른다.)
2. 텍스트와 패턴의 첫 글자부터 차례대로 비교를 시작한다.
3. 글자가 일치한다면 텍스트와 패턴의 인덱스를 모두 1씩 증가시킨다.
4. 글자가 불일치할 경우, 패턴의 인덱스가 0이 아니라면 LPS 테이블을 참조하여 패턴의 탐색 위치를 변경(건너뛰기)하고 다시 비교한다. 이때 텍스트의 인덱스는 뒤로 돌아가지 않는다.
5. 패턴의 인덱스가 0인데 불일치한다면, 텍스트의 인덱스만 1 증가시킨다.
6. 패턴의 마지막 글자까지 모두 일치했다면 '패턴 찾기'에 성공한 것이다. 찾은 위치를 기록한 뒤, 다음 탐색을 위해 LPS 테이블을 참조하여 패턴 인덱스를 조정하고 탐색을 계속 이어간다.

> **참고:** KMP 알고리즘의 핵심은 매칭이 실패했을 때 **'어디까지 매칭되었던 사실을 재활용할 수 있는지'** 파악하는 것이다. LPS 테이블이 이 역할을 수행하여 불필요한 중복 비교를 없애준다.

### 💻 KMP 알고리즘 기본 코드 (Python)

LPS 테이블을 만드는 과정(전처리)과 실제 텍스트에서 패턴을 찾는 과정으로 함수를 나누어 구현하는 것이 일반적이다.

```python
# 1. LPS(실패 함수) 테이블 생성 함수
def make_table(pattern):
    m = len(pattern)
    table = [0] * m
    j = 0 # 일치하는 접두사의 길이를 의미하는 포인터
    
    # i는 1부터 끝까지 진행하며 패턴 안에서 패턴을 찾음
    for i in range(1, m):
        # 글자가 다르면 j를 이전 일치했던 지점으로 되돌림
        while j > 0 and pattern[i] != pattern[j]:
            j = table[j - 1]
            
        # 글자가 같다면 j를 1 증가시키고 테이블에 기록
        if pattern[i] == pattern[j]:
            j += 1
            table[i] = j
            
    return table

# 2. KMP 탐색 함수
def kmp(text, pattern):
    table = make_table(pattern)
    result = [] # 패턴이 등장하는 시작 인덱스를 담을 리스트
    n = len(text)
    m = len(pattern)
    j = 0 # 패턴의 포인터
    
    # i는 텍스트의 포인터 (단방향으로만 전진)
    for i in range(n):
        # 글자가 다르면 j를 테이블을 참조해 되돌림
        while j > 0 and text[i] != pattern[j]:
            j = table[j - 1]
            
        # 글자가 같다면
        if text[i] == pattern[j]:
            # 패턴의 끝까지 모두 일치한 경우
            if j == m - 1:
                result.append(i - m + 1) # 패턴이 시작된 텍스트의 인덱스 기록
                j = table[j]             # 다음 매칭을 위해 j를 되돌림
            # 아직 패턴의 끝이 아니라면 j만 1 증가
            else:
                j += 1
                
    return result

```

### ⏱️ 시간 복잡도

KMP 알고리즘의 시간 복잡도는 **$O(N + M)$** 이다. ($N$: 텍스트의 길이, $M$: 패턴의 길이)
패턴의 길이가 $M$일 때 LPS 테이블을 만드는 데 $O(M)$의 시간이 걸리고, 길이가 $N$인 텍스트를 한 번 순회하며 탐색하는 데 $O(N)$의 시간이 걸린다. 만약 단순 무식하게 모든 경우의 수를 비교하는 브루트 포스(Brute Force) 방식을 사용했다면 $O(N \times M)$이 걸렸겠지만, KMP는 이를 선형 시간으로 단축해 준다.

### 📝 예시 문제: 찾기 (백준 1786번)

워드프로세서 등의 프로그램에서 '찾기' 기능을 구현하려고 한다. 문자열 $T$가 주어지고 문자열 $P$가 주어질 때, $T$ 중간에 $P$가 몇 번 나타나는지, 그리고 나타나는 위치가 어디인지 알아내는 프로그램을 작성하시오. (공백도 하나의 문자로 취급한다.)

* **입력:** 첫째 줄에 문자열 $T$가, 둘째 줄에 문자열 $P$가 주어진다. $T$와 $P$의 길이는 1 이상 1,000,000 이하이다.
* **출력:** 첫째 줄에 $T$ 중간에 $P$가 몇 번 나타나는지 출력한다. 둘째 줄에는 $P$가 나타나는 위치를 공백으로 구분해 출력한다. 단, 위치는 1부터 시작하는 1-based index를 기준으로 한다.

---

### 💡 풀이 접근법

입력되는 문자열의 길이가 최대 100만(1,000,000)이므로, 다중 for문을 이용한 단순 문자열 비교로는 100% 시간 초과가 발생한다. KMP 알고리즘을 사용하라고 만들어진 아주 클래식한 문제다.

* **주의할 점:** 공백을 포함한 전체 문자열을 그대로 읽어와야 한다. 파이썬의 `input()`이나 `sys.stdin.readline()`을 사용할 때 `strip()`을 잘못 사용하면 의도치 않게 양 끝의 공백이 날아가 버릴 수 있으므로, 줄 바꿈 문자(`\n`)만 제거하여 변수에 담아주는 것이 안전하다.
* 출력 조건에서 인덱스가 0이 아닌 1부터 시작한다고 명시했으므로, 찾은 인덱스에 1을 더해서 출력해야 한다.

---

### 💻 풀이 코드 (Python)

```python
import sys

# 공백이 포함될 수 있으므로 줄바꿈 문자('\n')만 제거하여 입력받음
text = sys.stdin.readline().replace("\n", "")
pattern = sys.stdin.readline().replace("\n", "")

def make_table(p):
    m = len(p)
    table = [0] * m
    j = 0
    for i in range(1, m):
        while j > 0 and p[i] != p[j]:
            j = table[j - 1]
        if p[i] == p[j]:
            j += 1
            table[i] = j
    return table

def kmp(t, p):
    table = make_table(p)
    result = []
    n = len(t)
    m = len(p)
    j = 0
    
    for i in range(n):
        while j > 0 and t[i] != p[j]:
            j = table[j - 1]
        
        if t[i] == p[j]:
            if j == m - 1:
                # 문제 조건에 맞게 1-based 인덱스로 변환하여 저장
                result.append(i - m + 2) 
                j = table[j]
            else:
                j += 1
                
    return result

# KMP 알고리즘 수행
matches = kmp(text, pattern)

# 1. 일치하는 패턴의 개수 출력
print(len(matches))
# 2. 일치하는 위치들을 띄어쓰기로 구분하여 출력
print(*matches)

```
