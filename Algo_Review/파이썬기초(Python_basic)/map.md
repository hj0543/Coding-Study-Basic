`map()` 함수는 파이썬으로 코딩 테스트를 볼 때 '숨 쉬는 것만큼 자연스럽게' 써야 하는 핵심 내장 함수다. 

쉽게 말해 **"리스트나 튜플 안에 있는 모든 원소들에 똑같은 함수(규칙)를 일괄적으로 싹 다 적용시켜 주는 요술 지팡이"**라고 생각하면 된다. `for`문을 돌면서 하나씩 값을 바꾸는 귀찮은 작업을 단 한 줄로 끝내버릴 수 있다.

### 💡 핵심 개념 및 동작 과정

* **형식:** `map(적용할 함수, 반복 가능한 데이터(리스트 등))`
* **특징:** `map`의 결과물은 리스트가 아니라 'map 객체(iterator)' 상태로 반환되기 때문에, 눈으로 확인하거나 인덱스로 접근하려면 반드시 `list()`나 `tuple()`로 껍데기를 씌워줘야 한다. (단, 변수 여러 개에 언패킹해서 담을 때는 생략 가능하다!)

### 💻 map 활용법 기본 코드 (Python)

코딩 테스트에서 `map`이 쓰이는 가장 대표적인 3가지 상황이다.

#### 1. 코딩 테스트 국민 입력 공식 (문자열 -> 정수 변환)

"10 20 30"처럼 공백으로 구분된 문자열 입력을 받아서, 각각을 정수(`int`)로 변환할 때 무조건 쓰인다.

```python
# 사용자가 "10 20"을 입력했다고 가정
# input().split() -> ["10", "20"] (문자열 리스트)
# map이 리스트 안의 모든 문자열에 int() 함수를 씌워줌
a, b = map(int, input().split()) 
print(a + b) # 결과: 30 (정수 연산 성공!)

# 원소가 많을 때는 list()로 묶어줌
numbers = list(map(int, input().split()))

```

#### 2. 리스트의 모든 원소 일괄 변환 (`for`문 대체)

실수(`float`) 리스트를 정수(`int`) 리스트로 바꾸거나, 문자열 리스트로 바꿀 때 아주 유용하다.

```python
floats = [1.1, 2.5, 3.9]
# map을 쓴다면? (단 한 줄!)
ints = list(map(int, floats))
print(ints) # 결과: [1, 2, 3]

# 숫자를 다시 문자열로?
strs = list(map(str, ints))
print(strs) # 결과: ['1', '2', '3']

```

#### 3. 앞서 배운 lambda와 환상의 콜라보

내가 원하는 복잡한 수식이나 규칙을 `lambda`로 만들어서 `map`에 집어넣으면, 배열 전체의 값을 순식간에 조작할 수 있다.

```python
arr = [1, 2, 3, 4, 5]
# 모든 원소에 10을 곱하기
multiplied = list(map(lambda x: x * 10, arr))
print(multiplied) # 결과: [10, 20, 30, 40, 50]

# 두 리스트의 같은 위치 원소끼리 더하기
arr1 = [1, 2, 3]
arr2 = [4, 5, 6]
sum_arr = list(map(lambda x, y: x + y, arr1, arr2))
print(sum_arr) # 결과: [5, 7, 9]

```

### ⏱️ 시간 복잡도

`map()` 함수의 시간 복잡도는 적용할 데이터의 개수가 $N$개일 때 **$O(N)$**이다. 파이썬 내부적으로 C언어로 최적화되어 구현되어 있기 때문에, 파이썬 코드상에서 `for`문을 돌며 `append()` 하는 것보다 메모리도 적게 쓰고 실행 속도도 미세하게 더 빠르다는 장점이 있다.
