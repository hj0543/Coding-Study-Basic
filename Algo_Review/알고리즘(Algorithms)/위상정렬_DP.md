위상 정렬(Topological Sort)과 다이나믹 프로그래밍(DP)의 결합은 코딩 테스트 그래프 분야의 단골 심화 출제 유형이다.

순서가 정해진 작업들을 차례대로 수행하면서, **'특정 작업까지 완료하는 데 걸리는 최소/최대 시간'**이나 **'특정 노드까지 가는 경로의 경우의 수'**를 구할 때 이 두 가지를 섞어 쓰면 완벽한 시너지를 낸다. 단순 위상 정렬이 '순서'만 찾아준다면, 여기에 DP를 얹어서 순서에 따른 '누적된 최적의 결괏값'까지 한 번에 계산해 내는 것이다.

### 💡 알고리즘 동작 과정

기본적인 뼈대는 큐(Queue)를 활용한 위상 정렬(Kahn's Algorithm)과 완전히 똑같다. 단지 노드를 꺼내고 다음 노드로 넘어갈 때 **'DP 테이블을 갱신하는 한 줄'**이 추가될 뿐이다.

1. 진입차수(In-degree) 배열과 그래프(인접 리스트)를 초기화한다.
2. 각 노드별 기본 소요 시간(또는 비용)을 담은 배열과, 누적된 최적의 값을 저장할 **DP 테이블**을 초기화한다. (보통 초기 DP 값은 자기 자신의 기본 시간으로 설정한다.)
3. 진입차수가 0인 노드를 큐에 넣는다.
4. 큐에서 노드(현재 노드)를 꺼내 인접한 다음 노드들을 확인한다.
5. **(DP 갱신 핵심)** 다음 노드에 도달하기 위한 누적 시간을 갱신한다.
* 점화식: `dp[next] = max(dp[next], dp[now] + cost[next])`
* *왜 최댓값(`max`)인가?* 여러 개의 선수 작업이 모두 끝나야만 다음 작업을 시작할 수 있는 경우, 가장 오래 걸리는 선수 작업의 시간에 맞춰야 하기 때문이다.


6. 다음 노드의 진입차수를 1 빼주고, 0이 되었다면 큐에 새롭게 넣는다.
7. 큐가 빌 때까지 4~6번 과정을 반복한다.

### 💻 위상 정렬 + DP 기본 코드 (Python)

```python
from collections import deque

# 노드 개수 v, 간선 개수 e
v, e = 4, 4
# 각 노드별 작업에 걸리는 기본 시간 (1번 노드부터 사용)
time = [0, 10, 20, 10, 100] 

indegree = [0] * (v + 1)
graph = [[] for _ in range(v + 1)]
dp = [0] * (v + 1) # 해당 노드까지 도달하는 데 걸리는 최대 누적 시간

# 간선 정보 입력 (예시 생략)
# graph[a].append(b)
# indegree[b] += 1

def topology_sort_dp():
    q = deque()
    
    # 1. 진입차수가 0인 노드를 큐에 삽입하고, DP 테이블의 초기값을 자기 자신의 시간으로 설정
    for i in range(1, v + 1):
        if indegree[i] == 0:
            q.append(i)
            dp[i] = time[i]
            
    while q:
        now = q.popleft()
        
        for nxt in graph[now]:
            # 2. DP 테이블 갱신 (핵심 로직)
            # 다음 노드까지 걸리는 시간은 '기존에 계산된 시간'과 '현재 노드를 거쳐서 가는 시간' 중 최댓값
            dp[nxt] = max(dp[nxt], dp[now] + time[nxt])
            
            # 3. 진입차수 감소 및 큐 삽입
            indegree[nxt] -= 1
            if indegree[nxt] == 0:
                q.append(nxt)

topology_sort_dp()
print(dp) # 각 노드별로 작업이 완료되는 데 필요한 최소 시간들이 담겨 있음

```

### ⏱️ 시간 복잡도

위상 정렬의 시간 복잡도와 동일하게 **$O(V + E)$** 이다. ($V$: 정점의 개수, $E$: 간선의 개수)
모든 정점과 간선을 한 번씩만 확인하면서 DP 배열을 상수 시간($O(1)$)에 갱신하기 때문에, 데이터가 10만 개 단위로 주어져도 매우 빠르게 동작한다.

### 📝 예시 문제: ACM Craft (백준 1005번)

서기 2012년! 드디어 2년간 수많은 국민들을 기다리게 한 게임 ACM Craft (ACM 크래프트)가 발매되었다.
이 게임은 건물을 짓는 순서가 정해져 있다. 예를 들어 1번 건물을 지어야만 2번 건물을 지을 수 있고, 2번과 3번 건물을 모두 지어야만 4번 건물을 지을 수 있다.
각 건물을 짓는 데 걸리는 시간이 주어질 때, 특정 건물 $W$를 건설하기 위해 필요한 최소 시간을 구하는 프로그램을 작성하시오.

* **입력:** 첫째 줄에 테스트 케이스의 개수 $T$가 주어진다. 각 테스트 케이스의 첫째 줄에는 건물의 개수 $N$ ($1 \le N \le 1000$)과 건물 건설 순서 규칙의 총개수 $K$ ($1 \le K \le 100,000$)가 주어진다. 둘째 줄에는 각 건물에 걸리는 시간 $D$가 주어진다. 셋째 줄부터 $K$개의 줄에는 건설 순서 $X, Y$ ($X$를 지은 다음 $Y$를 짓는 것이 가능)가 주어진다. 마지막 줄에는 승리하기 위해 지어야 할 건물 번호 $W$가 주어진다.
* **출력:** 건물 $W$를 건설완료 하는 데 드는 최소 시간을 출력한다.

---

### 💡 풀이 접근법

위상 정렬과 DP가 결합된 문제 중 **가장 유명하고 교과서적인 문제**다.

'모든 선수 건물이 지어져야만 다음 건물을 지을 수 있다'는 조건 때문에, 여러 개의 진입 간선이 있을 경우 **가장 건설이 오래 걸린 트리의 시간**을 따라가야 한다.
즉, $Y$ 건물을 짓기 직전까지 걸리는 누적 시간은, $Y$로 들어오는 모든 $X$ 건물들의 누적 시간 중 '최댓값'이 되어야 한다. 앞서 정리한 뼈대 코드의 `max` 점화식을 그대로 적용하면 완벽하게 풀린다.

---

### 💻 풀이 코드 (Python)

```python
import sys
from collections import deque
input = sys.stdin.readline

t = int(input())

for _ in range(t):
    n, k = map(int, input().split())
    # 각 건물을 짓는 데 걸리는 시간 (인덱스 맞추기 위해 앞에 0 추가)
    times = [0] + list(map(int, input().split()))
    
    graph = [[] for _ in range(n + 1)]
    indegree = [0] * (n + 1)
    # 해당 건물까지 지어지는 데 걸리는 총 누적 시간
    dp = [0] * (n + 1)
    
    # 건설 순서 규칙 입력
    for _ in range(k):
        u, v = map(int, input().split())
        graph[u].append(v)
        indegree[v] += 1
        
    w = int(input()) # 최종적으로 지어야 할 목표 건물 번호
    
    q = deque()
    
    # 초기 세팅: 진입차수가 0인(선행 조건이 없는) 건물부터 큐에 삽입
    for i in range(1, n + 1):
        if indegree[i] == 0:
            q.append(i)
            dp[i] = times[i] # 누적 시간의 초기값은 자기 자신의 건설 시간
            
    # 위상 정렬 + DP 수행
    while q:
        now = q.popleft()
        
        # 목표 건물 w의 건설이 끝났다면 더 볼 필요 없이 종료 (최적화)
        if now == w:
            break
            
        for nxt in graph[now]:
            # nxt 건물을 짓기 전까지의 시간 갱신 (가장 늦게 끝나는 선행 건물 기준)
            dp[nxt] = max(dp[nxt], dp[now] + times[nxt])
            indegree[nxt] -= 1
            
            if indegree[nxt] == 0:
                q.append(nxt)
                
    # 목표 건물 w를 짓는 데 걸린 최소(실질적으론 누적된 최댓값) 시간 출력
    print(dp[w])

```
