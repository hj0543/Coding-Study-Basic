에라토스테네스의 체(Sieve of Eratosthenes)는 특정 범위 내의 **모든 소수(Prime Number)를 대량으로 빠르고 정확하게 구하는 대표적인 알고리즘**이다.

마치 체(Sieve)로 가루를 걸러내듯이, 가장 작은 소수부터 시작해 그 소수의 배수들을 차례대로 지워나가는 직관적인 방식을 사용한다. 단일 숫자의 소수 판별은 제곱근까지만 나누어보는 방식($O(\sqrt{N})$)으로 충분하지만, 1부터 $N$까지의 '모든' 소수를 한 번에 찾아야 할 때는 이 알고리즘이 압도적으로 유리하다.

### 💡 알고리즘 동작 과정

1. 2부터 $N$까지의 모든 자연수를 나열한다. (보통 리스트의 인덱스를 활용하기 위해 `True`로 초기화된 배열을 만든다.)
2. 아직 지워지지 않은 수 중에서 가장 작은 수 $i$를 찾는다. (이 수는 소수다.)
3. 자기 자신 $i$를 제외한 $i$의 모든 배수를 배열에서 지운다. (`False`로 변경)
4. $i$가 $\sqrt{N}$이 될 때까지 2~3번 과정을 반복한다.

> **참고:** $\sqrt{N}$까지만 검사하는 이유는, $N$의 약수는 대칭적으로 짝을 이루기 때문에 $\sqrt{N}$보다 큰 수의 배수는 이미 그 이전의 작은 소수들에 의해 모두 지워졌기 때문이다. 따라서 $\sqrt{N}$까지만 배수를 지워도 남은 수들은 자연스럽게 모두 소수가 된다.

---

### 💻 에라토스테네스의 체 기본 코드 (Python)

```python
import math

n = 1000 # 2부터 1000까지의 모든 소수 구하기

# 처음엔 모든 수가 소수(True)인 것으로 초기화 (0과 1은 제외하므로 넉넉히 n+1 크기로 생성)
sieve = [True] * (n + 1)
sieve[0] = False
sieve[1] = False

# 2부터 n의 제곱근까지만 확인
for i in range(2, int(math.sqrt(n)) + 1):
    if sieve[i] == True: # i가 소수인 경우
        # i를 제외한 i의 모든 배수를 지우기
        # i*2 부터 시작해도 되지만, i*i 부터 시작하는 것이 중복을 줄여 더 효율적임
        for j in range(i * i, n + 1, i):
            sieve[j] = False

# 모든 소수 출력 (sieve[i]가 True인 인덱스 i 출력)
for i in range(2, n + 1):
    if sieve[i]:
        print(i, end=' ')

```

### ⏱️ 시간 복잡도

에라토스테네스의 체의 시간 복잡도는 **$O(N \log(\log N))$** 이다.
배수들을 지워나가는 연산이 반복될수록 검사해야 할 숫자가 급격히 줄어들기 때문에, 사실상 선형 시간 $O(N)$에 가까울 정도로 매우 빠르게 동작한다. 다만, 탐색 범위 $N$의 크기만큼 리스트를 메모리에 할당해야 하므로 **메모리 공간이 많이 필요하다는 단점**이 있다. (보통 코딩 테스트에서는 $N$이 1,000만 이하일 때까지만 사용을 권장한다.)

### 📝 예시 문제: 소수 구하기 (백준 1929번)

$M$이상 $N$이하의 소수를 모두 출력하는 프로그램을 작성하시오.

* **입력:** 첫째 줄에 자연수 $M$과 $N$이 빈칸을 사이에 두고 주어진다. ($1 \le M \le N \le 1,000,000$) $M$이상 $N$이하의 소수가 하나 이상 있는 입력만 주어진다.
* **출력:** 한 줄에 하나씩, 증가하는 순서대로 소수를 출력한다.

---

### 💡 풀이 접근법

$N$이 최대 1,000,000(100만)이므로, 각 숫자를 일일이 소수 판별 함수에 넣어서 검사하면 최악의 경우 시간 초과가 날 수 있다. 대량의 소수를 판별해야 하므로 전형적인 **에라토스테네스의 체** 문제다.

* 100만 크기의 배열을 만들어 에라토스테네스의 체를 돌려 소수 배열을 완성한다.
* 이후 $M$부터 $N$까지 순회하며 값이 `True`인 인덱스만 출력해 주면 된다. (1은 소수가 아니므로 초기화 때 반드시 `False`로 명시해 두어야 반례를 피할 수 있다.)

---

### 💻 풀이 코드 (Python)

```python
import sys
import math
input = sys.stdin.readline

m, n = map(int, input().split())

# 에라토스테네스의 체 초기화
sieve = [True] * (n + 1)
sieve[0] = False
sieve[1] = False

# 2부터 n의 제곱근까지만 모든 배수 지우기
for i in range(2, int(math.sqrt(n)) + 1):
    if sieve[i] == True:
        # 파이썬의 range(start, stop, step)을 활용하여 i의 배수들을 False로 변환
        for j in range(i * i, n + 1, i):
            sieve[j] = False

# m부터 n까지 소수만 출력
for i in range(m, n + 1):
    if sieve[i]:
        print(i)

```
