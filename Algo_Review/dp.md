복잡한 문제를 더 작은 하위 문제로 나누어 해결하는 알고리즘 기법인 **동적 계획법(Dynamic Programming, DP)**

### 💡 DP의 특징 및 동작 과정

DP는 주로 최적화 문제나 경우의 수를 구하는 문제에서 사용되며, 다음 두 가지 조건을 만족할 때 진가를 발휘한다.

1. **부분 문제 중복 (Overlapping Subproblems):** 동일한 작은 하위 문제들이 반복해서 나타난다.
2. **최적 부분 구조 (Optimal Substructure):** 부분 문제의 최적 결과값을 조합해 전체 문제의 최적 결과를 낼 수 있다.

DP는 계산 결과를 저장해 두었다가 재활용하여 속도를 높이는 것이 핵심이며, 구현 방식은 크게 두 가지다.

* **메모이제이션 (Memoization, Top-Down):** 재귀 함수를 사용하여 위에서 아래로 문제를 해결한다. 한 번 푼 문제의 결과는 리스트나 딕셔너리에 기록(메모)해 두고, 다시 필요할 때 꺼내 쓴다.
* **타뷸레이션 (Tabulation, Bottom-Up):** 반복문을 사용하여 아래에서 위로 문제를 해결한다. 가장 작은 부분 문제부터 차례대로 답을 구해 배열(DP 테이블)을 채워나간다. (파이썬에서는 재귀 깊이 제한 이슈가 있어 주로 이 방식을 권장한다.)

---

### 💻 DP 기본 사용법 (Python)

가장 대표적인 DP 예시인 피보나치 수열을 **Bottom-Up(반복문)** 방식으로 구현하는 기본 문법이다. 이전의 계산 결과를 배열에 저장하여 중복 연산을 방지한다.

```python
# 1. DP 테이블 초기화 (보통 구하려는 값의 최대 크기만큼 0으로 생성)
n = 10
dp = [0] * (n + 1)

# 2. 가장 작은 부분 문제의 초기값 설정
dp[1] = 1
dp[2] = 1

# 3. 반복문을 이용해 점화식(상태 전이 방정식) 적용
for i in range(3, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]

print(dp[n])  # 출력: 55 (10번째 피보나치 수)

```

### ⏱️ 시간 복잡도

일반적인 재귀 함수로 피보나치 수열을 구하면 동일한 연산을 반복하여 $O(2^N)$의 기하급수적인 시간이 걸린다.    
하지만 DP를 사용하여 한 번 계산한 값을 저장해두면, 각 부분 문제를 한 번씩만 풀게 되므로 시간 복잡도를 $O(N)$으로 획기적으로 단축할 수 있다.

### 📝 예시 문제: 1로 만들기 (백준 1463번)

정수 X에 사용할 수 있는 연산은 다음과 같이 세 가지 이다.

1. X가 3으로 나누어 떨어지면, 3으로 나눈다.
2. X가 2로 나누어 떨어지면, 2로 나눈다.
3. 1을 뺀다.

정수 N이 주어졌을 때, 위와 같은 연산 세 개를 적절히 사용해서 1을 만들려고 한다. 연산을 사용하는 횟수의 최솟값을 출력하시오.

* **입력:** 첫째 줄에 1보다 크거나 같고, 10^6보다 작거나 같은 정수 N이 주어진다.
* **출력:** 첫째 줄에 연산을 하는 횟수의 최솟값을 출력한다.

---

### 💡 풀이 접근법

이 문제는 그리디(Greedy)하게 단순히 큰 수(3)로 먼저 나눈다고 최적해가 나오지 않습니다. (예: 10의 경우 10 -> 5 -> 4 -> 2 -> 1 (4번) 보다 10 -> 9 -> 3 -> 1 (3번) 이 더 빠름). 따라서 모든 경우를 고려하되 중복 계산을 막는 **DP**를 사용해야 완벽하게 풀립니다.

* **DP 테이블 정의:** `dp[i]` = 정수 i를 1로 만드는 데 필요한 최소 연산 횟수
* **점화식(상태 전이 방정식) 세우기:**
* 기본적으로 1을 빼는 연산을 수행한다고 가정: $dp[i] = dp[i - 1] + 1$
* 2로 나누어 떨어지는 경우, 위 결과와 2로 나눈 결과 중 최솟값 선택: $dp[i] = \min(dp[i], dp[i // 2] + 1)$
* 3으로 나누어 떨어지는 경우, 위 결과와 3으로 나눈 결과 중 최솟값 선택: $dp[i] = \min(dp[i], dp[i // 3] + 1)$



---

### 💻 풀이 코드 (Python)

```python
import sys
input = sys.stdin.readline

def make_one():
    n = int(input())
    
    # 1. DP 테이블 초기화 (0부터 N까지 기록하기 위해 N+1 크기 할당)
    dp = [0] * (n + 1)
    
    # 2. Bottom-Up 방식으로 배열 채우기
    # dp[1]은 1을 1로 만드는 횟수이므로 0 (초기값으로 이미 0이 들어있음)
    for i in range(2, n + 1):
        # 먼저 1을 빼는 경우의 연산 횟수를 저장
        dp[i] = dp[i - 1] + 1
        
        # 현재 수가 2로 나누어 떨어지는 경우, 더 적은 횟수를 갱신
        if i % 2 == 0:
            dp[i] = min(dp[i], dp[i // 2] + 1)
            
        # 현재 수가 3으로 나누어 떨어지는 경우, 더 적은 횟수를 갱신
        if i % 3 == 0:
            dp[i] = min(dp[i], dp[i // 3] + 1)
            
    # N을 1로 만드는 최소 연산 횟수 출력
    print(dp[n])

# 함수 실행
make_one()

```
