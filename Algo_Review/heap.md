파이썬에서 최솟값이나 최댓값을 빠르게 찾아야 하거나 우선순위 큐(Priority Queue)를 구현할 때는 주로 내장 모듈인 **`heapq`**를 사용한다.    
`heapq`는 이진 트리 기반의 **최소 힙(Min Heap)** 자료구조를 제공한다.

### 💡 힙(Heap)의 특징 및 동작 과정

1. 파이썬의 `heapq`는 기본적으로 부모 노드의 값이 자식 노드의 값보다 항상 작거나 같은 **최소 힙(Min Heap)** 구조를 유지한다. 따라서 트리의 루트(인덱스 0)에는 항상 가장 작은 값이 위치한다.
2. **데이터 삽입 (`heappush`):** 배열의 맨 끝에 새로운 데이터를 추가한 뒤, 부모 노드와 값을 비교하며 조건(부모 $\le$ 자식)을 만족할 때까지 위로 끌어올린다.
3. **데이터 삭제 (`heappop`):** 루트 노드(최솟값)를 꺼낸다. 그 후 트리의 맨 마지막 노드를 루트 자리로 옮기고, 자식 노드들과 비교하며 아래로 끌어내려 힙 구조를 다시 맞춘다.
4. **리스트 변환 (`heapify`):** 이미 데이터가 들어있는 일반 리스트를 즉시 힙 구조로 재배열한다.

> **참고:** 파이썬은 최대 힙(Max Heap)을 기본으로 제공하지 않는다.   
> 만약 최대 힙이 필요하다면, 데이터를 넣을 때 부호를 반대로 바꾸어 `heapq.heappush(heap, -item)` 형태로 넣고, 뺄 때 다시 부호를 바꾸는 방식을 사용해야 한다.

---

### 💻 heapq 기본 사용법 (Python)

가장 직관적으로 많이 쓰이는 `heapq`의 기본 문법이다. `heapq`는 별도의 힙 객체를 만드는 것이 아니라, 일반적인 리스트를 힙처럼 다룰 수 있게 도와주는 함수들의 모음이다.

```python
import heapq

# 빈 리스트를 생성하여 힙으로 사용
heap = []

# 1. 힙에 원소 추가 (heappush)
heapq.heappush(heap, 4)
heapq.heappush(heap, 1)
heapq.heappush(heap, 7)
heapq.heappush(heap, 3)

# 2. 힙에서 원소 삭제 및 반환 (heappop)
# 가장 작은 값인 1이 반환되고, 힙 구조가 다시 정렬됨
min_value = heapq.heappop(heap)
print(min_value)  # 출력: 1

# 3. 기존 리스트를 힙으로 변환 (heapify)
data = [5, 2, 8, 1, 9]
heapq.heapify(data)
print(data[0])  # 출력: 1 (최솟값이 0번 인덱스에 위치하게 됨)

```

### ⏱️ 시간 복잡도

힙의 시간 복잡도는 삽입과 삭제 모두 **$O(\log N)$** 다. (N: 힙에 들어있는 원소의 개수)
일반 리스트에서 `min()` 함수를 사용해 최솟값을 찾으면 $O(N)$의 시간이 걸리지만, 힙을 사용하면 트리의 높이만큼만 탐색하므로 데이터가 많을수록 속도 차이가 압도적으로 빠르다.

### 📝 예시 문제: 최소 힙 (백준 1927번)

널리 잘 알려진 자료구조 중 최소 힙이 있다. 최소 힙을 이용하여 다음과 같은 연산을 지원하는 프로그램을 작성하시오.

1. 배열에 자연수 x를 넣는다.
2. 배열에서 가장 작은 값을 출력하고, 그 값을 배열에서 제거한다.

* **입력:** 첫째 줄에 연산의 개수 N($1 \le N \le 100,000$)이 주어진다. 다음 N개의 줄에는 연산에 대한 정보를 나타내는 정수 x가 주어진다. 만약 x가 자연수라면 배열에 x라는 값을 넣는(추가하는) 연산이고, x가 0이라면 배열에서 가장 작은 값을 출력하고 그 값을 배열에서 제거하는 경우이다.
* **출력:** 입력에서 0이 주어진 횟수만큼 답을 출력한다. 만약 배열이 비어 있는 경우인데 가장 작은 값을 출력하라고 한 경우에는 0을 출력하면 된다.

---

### 💡 풀이 접근법

이 문제는 최솟값을 지속적으로 찾아야 하므로 **최소 힙**을 사용하기에 완벽한 조건입니다. 파이썬의 `heapq` 모듈을 그대로 가져다 쓰면 시간 초과 없이 간단하게 해결할 수 있습니다.

* **입력이 자연수일 때:** `heapq.heappush()`를 사용해 힙에 값을 넣습니다.
* **입력이 0일 때:** 힙이 비어있지 않다면 `heapq.heappop()`으로 최솟값을 빼서 출력하고, 힙이 비어있다면 `0`을 출력합니다.

---

### 💻 풀이 코드 (Python)

```python
import sys
import heapq
input = sys.stdin.readline

def min_heap_operations():
    heap = [] # 최소 힙으로 사용할 빈 리스트 생성

    # 연산의 개수 N 입력받기
    n = int(input())

    for _ in range(n):
        x = int(input()) # 연산 정보 입력

        if x == 0:
            # 1. x가 0일 때: 배열에서 가장 작은 값을 출력하고 제거
            if heap: # 힙에 원소가 존재한다면
                # 가장 작은 값을 꺼내서 출력
                print(heapq.heappop(heap))
            else:
                # 힙이 비어있는데 출력을 요구하면 0 출력
                print(0)
        else:
            # 2. x가 자연수일 때: 배열에 x 값을 넣음
            heapq.heappush(heap, x)

# 최소 힙 함수 실행
min_heap_operations()

```
