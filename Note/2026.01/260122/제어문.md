
# 📝 파이썬 제어문 (Control Statement)

코드의 실행 흐름을 제휘하는 지휘자와 같다. 조건에 따라 실행하거나, 특정 코드를 반복하도록 명령한다.

## 1. 조건문 (Conditional Statement)

주어진 조건이 **참(True)**인지 **거짓(False)**인지에 따라 서로 다른 코드를 실행한다.

### 구조와 순서

* **`if`**: 가장 처음 조건을 검사한다. (필수)
* **`elif`**: `if`가 거짓일 때, 추가적인 조건을 검사한다. (선택, 여러 개 가능)
* **`else`**: 모든 조건이 거짓일 때 실행된다. (선택)

### 💡 조건문 작성 시 주의할 점 (순서)

조건문은 위에서부터 순서대로 검사하기 때문에, **범위가 좁거나 중요한 조건을 상단에 배치**해야 한다.

```python
dust = 155

# 잘못된 예시 (논리 오류)
if dust > 30:
    print('보통')      # 155는 30보다 크므로 여기서 걸리고 끝난다.
elif dust > 150:
    print('매우 나쁨') # 실행될 기회가 없다.

# 올바른 예시
if dust > 150:
    print('매우 나쁨') # 155는 여기서 올바르게 걸러진다.
elif dust > 30:
    print('보통')

```

## 2. 반복문 (Loop Statement)

특정 코드 블록을 여러 번 반복해서 실행한다. 파이썬은 주로 `for`와 `while`을 사용한다.

### ① for 문

**횟수가 정해져 있거나**, 리스트 같은 **데이터 덩어리(Iterable)를 하나씩 꺼내 쓸 때** 유용하다.

* **구조**: `for 변수 in 반복_가능한_객체:`
* **작동 원리**: 리스트 등의 요소가 하나씩 변수에 할당되며 코드가 실행된다.

**다양한 순회 방식**

1. **리스트 순회**: 요소를 직접 꺼낸다.
2. **문자열 순회**: 문자 한 글자씩 꺼낸다.
3. **딕셔너리 순회**: **Key**가 순회된다. (Value는 `dict[key]`로 접근)
4. **인덱스 순회 (`range`)**: 리스트의 값을 수정해야 할 때 주로 사용한다.

```python
numbers = [4, 6, 10]

# 인덱스로 접근하여 원본 수정하기
for i in range(len(numbers)): 
    numbers[i] = numbers[i] * 2

print(numbers) # [8, 12, 20]

```

### ② while 문

**조건이 참인 동안** 계속 반복한다. **반복 횟수가 명확하지 않을 때** (예: 사용자가 '종료'라고 입력할 때까지) 사용한다.

* **주의**: 종료 조건(거짓이 되는 순간)을 만들지 않으면 **무한 루프**에 빠진다.

```python
number = 0
while number >= 0: # 양수면 계속 반복
    number = int(input("양수를 입력하세요 (음수 입력 시 종료): "))
print("종료되었습니다.")

```

## 3. 반복 제어 (Loop Control)

반복문을 돌다가 특정 상황에서 흐름을 바꾸고 싶을 때 사용한다.

### 비교 요약

| 키워드 | 역할 | 비유 |
| --- | --- | --- |
| **break** | 반복을 **즉시 종료**하고 탈출한다. | "여기까지! 퇴근해!" |
| **continue** | 밑에 코드는 무시하고 **다음 반복**으로 넘어간다. | "이번 건 패스, 다음 것 가져와!" |
| **pass** | **아무것도 하지 않는다.** (문법적 오류 방지용) | "일단 빈 칸으로 둬." |

### 예시 코드로 비교

```python
# 1. break (반복 종료)
for i in range(10):
    if i == 3:
        break
    print(i) 
# 출력: 0, 1, 2 (3을 만나면 아예 끝남)

# 2. continue (이번 턴만 건너뜀)
for i in range(5):
    if i == 3:
        continue
    print(i)
# 출력: 0, 1, 2, 4 (3만 빼고 출력)

# 3. pass (그냥 지나감)
for i in range(5):
    if i == 3:
        pass # 아무 일도 안 함. 그냥 다음 줄로 감.
    print(i)
# 출력: 0, 1, 2, 3, 4 (모두 출력)

```

## 💡 팁: 중첩 반복문 (Nested Loops)

반복문 안에 반복문이 있는 형태이다.

* **작동 순서**: 바깥쪽 루프 1회 수행 -> 안쪽 루프 전체 수행 -> 바깥쪽 루프 다음 회차...
* `break`를 사용하면, **자신이 속한 가장 가까운 반복문 하나만** 탈출한다.

```python
for outer in ['A', 'B']:
    for inner in ['c', 'd']:
        print(outer, inner)
# A c, A d (안쪽 끝) -> B c, B d (바깥쪽 다음) 순서로 출력

```
